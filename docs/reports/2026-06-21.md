### 1. AutoPass: Evidence-Guided LLM Agents for Compiler Performance Tuning

**Intro:** 編譯器優化對於提升軟體在現代系統上的效能至關重要，但現有的優化方法多依賴固定的優化階段序列與參數設定，無法針對不同程式與硬體達到最佳效果。傳統的自動調優方法雖能探索優化空間，但計算成本高且缺乏解釋性。大型語言模型（LLMs）在程式碼生成上展現潛力，但直接應用於執行效能調優困難，因為效能受微架構複雜交互與噪聲測量影響。本文提出AutoPass，一個多代理框架，結合編譯器內部訊息與執行時回饋，利用LLM進行編譯器優化決策，並在LLVM上實作，針對x86-64與ARM64平台進行評估。

**Motivation:** 現有的Profile-Guided Optimization（PGO）方法受限於分析的局部性與噪聲，往往保守且錯失重要優化機會。黑盒式的自動調優在有限的編譯與執行預算下難以找到有效的優化序列。優化效能需要考慮程式結構、控制流與硬體特性間的複雜交互，這超出傳統靜態分析能力。LLM具備語意推理能力，能理解程式語意與優化上下文，但缺乏實際執行回饋導致優化決策不穩定。AutoPass透過多代理系統，結合編譯器內部狀態與執行時數據，實現閉環優化，克服上述限制。

**Method:** AutoPass架構包含四個主要代理：Score Agent負責從程式中識別優化關鍵函式並篩選；Analysis Agent解析LLVM中介表示與編譯器診斷訊息，產生結構化優化狀態；Reasoning Agent根據分析結果與執行回饋，生成並修正優化通道序列與參數設定；Evaluation Agent負責編譯、執行並收集效能數據，提供回饋以指導下一輪優化。系統在LLVM基礎上運作，利用有限次數的編譯執行迭代，透過語意推理與證據導向的決策，逐步改進優化策略，並採用語法檢查與修復確保生成的優化序列合法。

**Result:** 在x86-64與ARM64兩種硬體平台，針對包含cBench、PolyBench、CoreMark等多樣化基準測試，AutoPass在嚴格的執行迭代預算下，平均分別達到1.043×與1.117×的幾何平均加速，優於Instrumentation PGO、AutoFDO、CSSPGO及OpenTuner等基線方法。實驗顯示迭代回饋顯著提升優化穩定性與效能，Score Agent在函式篩選上優於PGO熱點選擇。架構敏感性分析表明AutoPass能根據目標硬體調整優化策略。消融實驗指出Reasoning Agent對效能提升最為關鍵，Evaluation Agent提升穩定性，Analysis Agent加速收斂。多模型測試證明框架具備良好泛化性與模型無關性。

**Conclusion:** AutoPass提出一種結合LLM推理與編譯器內部證據及執行回饋的多代理優化框架，實現LLVM編譯器的效能調優。該系統在無需離線訓練與特定任務微調的條件下，能在有限迭代預算內穩定提升執行效能，並提供可解釋的優化決策。實驗結果證明其優於傳統PGO與黑盒自動調優方法，且能適應不同硬體架構。此工作展示了結合現有編譯器基礎設施與LLM推理的可行性，為軟體工程領域帶來實用且具解釋性的自適應優化流程。

**Research Gap:** 本文雖展示AutoPass在多平台與多基準上的優異表現，但仍存在幾點不足。首先，實驗迭代次數受限於三輪，對於更大規模或更複雜程式的收斂性與效能提升潛力未充分驗證。其次，對於生成優化序列的成本與實際部署風險（如編譯時間增加、記憶體消耗）未有詳細討論。第三，系統對於失敗案例的處理與泛化能力尚缺乏深入分析，尤其在面對極端或非典型程式時的穩定性未知。第四，雖有多模型測試，但不同LLM在推理品質與成本上的差異及其對整體系統效能的影響未完全展開。最後，全文未明確說明在多代理間協調機制的細節與潛在瓶頸，未來可加強對代理間互動與錯誤傳遞的研究。

- 入選理由：LLM rank #2 / 8/10: Theoretical novelty in proving that a single fixed RNN can approximate any continuous function by running longer, supported by minimax lower bounds, offering a strong long-term research signal in neural approximation the

**Intro:** 本論文探討固定結構的 ReLU 循環神經網路（RNN）是否能透過延長運算時間，逼近任意連續函數。傳統神經網路逼近理論通常需為不同精度建立不同網路，限制了實際部署的靈活性。作者提出一種新穎的計算模型 TMNU（Turing Machine with Neural Units），結合圖靈機的離散控制與神經網路的連續變換，作為分析 RNN 逼近能力的中介工具。論文證明對於定義在區間[-1,1]上的任意連續函數，存在一個固定大小與權重的 RNN，隨著運算時間增加，其輸出能均勻收斂至目標函數。此結果擴展了先前僅針對多項式函數的研究，並提供了明確的隱藏層維度與權重範圍界限。

**Motivation:** 在工程與產品層面，固定結構且可透過延長運算時間提升精度的模型，能大幅降低模型部署與更新成本，尤其適合資源受限的裝置。研究上，傳統逼近理論無法解決隨精度變化需重新訓練或設計網路的問題，限制了理論與實務的連結。作者透過 TMNU 模型，克服了 RNN 與圖靈機間的連續性與離散性差異，提供一條可行路徑來證明固定 RNN 可逼近任意連續函數，對理解 RNN 的計算能力與逼近極限具有重要意義。

**Method:** 核心方法是引入 TMNU 模型，該模型結合圖靈機的離散狀態與帶 ReLU 激活的連續神經狀態。TMNU 透過有限狀態控制與分段線性變換操作神經狀態，能模擬多項式逼近與切比雪夫多項式展開。作者證明 TMNU 在神經狀態有界的條件下，可被深度 ReLU 神經網路模擬，進而由深度神經網路構造對應的 RNN。論文中詳細設計了多層次子程序結構，實現多項式與切比雪夫多項式的逐步逼近，並分析了逼近誤差隨運算時間的衰減速率。最後，透過度量熵理論證明此逼近速率在固定隱藏維度與權重範圍下接近最優。

**Result:** 論文證明對任意連續函數 f : [-1,1]→R，存在固定隱藏層維度（約 5320）與權重範圍的 RNN，使得其輸出隨運算時間趨近無限時，均勻收斂至 f。對多項式函數，逼近誤差以指數速率衰減，且此速率在度量熵意義下為最優。對更廣泛的 Dini-Lipschitz 連續函數，逼近誤差衰減速率與函數切比雪夫展開的收斂速率直接相關。實驗結果明確界定了逼近誤差、隱藏層大小、權重範圍與運算時間間的關係，並透過下界證明運算時間是不可避免的資源消耗。

**Conclusion:** 本論文成功建立固定結構 RNN 透過延長運算時間逼近任意連續函數的理論基礎，並提出 TMNU 作為分析工具，連結圖靈機與 RNN 的計算能力。此結果對於理解 RNN 的理論極限及其在實務中以固定模型應對多種精度需求具有指導意義。限制在於隱藏層維度較大，且全文未明確說明實際訓練過程與泛化能力，此外，逼近速率依賴函數的光滑性，對非 Dini-Lipschitz 函數的逼近尚未涵蓋。

**Research Gap:** 論文中隱藏層維度固定且較大（約 5320），對於實際部署在資源有限設備的可行性與效率未充分探討。逼近理論依賴函數的 Dini-Lipschitz 連續性，對於不滿足此條件的函數類型未提供分析或實驗。全文未明確說明如何從理論構造轉化為可訓練的模型參數，缺乏訓練策略與泛化性能的驗證。此外，對於逼近失敗案例、噪聲魯棒性及計算成本的實際影響未有深入討論。消融實驗與與其他逼近方法的比較亦不足，限制了對方法優劣的全面評估。

### 3. Shifting-based Optimizable Linear Relaxations for General Activation Functions

- 分數：0
- arXiv: [2606.20292v1](http://arxiv.org/abs/2606.20292v1)
- 發表日期：2026-06-18
- 分類：cs.LG, cs.LO
- 入選理由：LLM rank #3 / 8/10: SLiR introduces a broadly applicable, optimization-friendly linear relaxation method for general activation functions, improving verification capabilities significantly, with credible experiments and practical usefulness

**Intro:** 神經網路在安全與關鍵領域的應用日益增加，對其行為提供形式化保證成為重要課題。許多驗證方法依賴可優化的線性鬆弛來近似激活函數，但現有技術多依賴為每種激活函數手工設計鬆弛，難以擴展至新興複雜激活函數。本文提出SLiR方法，透過斜率參數化及位移計算，實現對一般激活函數的可優化線性鬆弛，降低手工設計負擔並提升驗證效率。

**Motivation:** 現代神經網路越來越多使用複雜激活函數如GELU、SWISH、MISH等，這些函數缺乏簡單的閉式鬆弛表達式，導致現有驗證方法難以直接應用或需高昂手工調整。傳統方法在每次優化迭代都需重新驗證鬆弛的正確性，計算成本高昂且不利於梯度優化。SLiR透過分離斜率優化與鬆弛正確性計算，支援廣泛激活函數且能有效整合現有邊界傳播框架，對工程與產品落地有實際價值。

**Method:** SLiR核心在於將激活函數的線性鬆弛參數化為斜率，並透過位移操作計算對應的偏移量以確保鬆弛在輸入區間內的上下界正確性。對於可計算導數反函數的激活函數，SLiR可直接求解最大偏移量；對無閉式解的函數，則先以分段線性近似替代激活函數，再利用改良的Piyavskii Lipschitz優化方法計算安全的上下包絡線。此外，SLiR限制斜率參數於激活函數凸包的斜率範圍內，並以sigmoid函數平滑映射參數，提升優化穩定性與收斂速度。

**Result:** SLiR在多個基準測試中驗證了其優越性，包括MNIST與CIFAR10的卷積神經網路，涵蓋多種複雜激活函數。與α-CROWN等現有方法相比，SLiR在初始化階段即能驗證更多性質，優化後驗證數量提升顯著，最高達7.8倍。實驗也顯示SLiR在無閉式解函數上仍能保持良好性能，且改良的Piyavskii優化與斜率範圍限制對驗證效果與效率均有正面影響。消融實驗確認各項改進均有助於提升驗證數量與穩定性。

**Conclusion:** 本文提出的SLiR方法提供一種通用且可優化的線性鬆弛框架，能自動生成對多種激活函數有效且正確的鬆弛，降低專家手工設計負擔。SLiR結合斜率參數化與位移計算，支援梯度優化並整合現有邊界傳播驗證流程，實驗證明其在多種激活函數與網路架構上均優於現有方法。限制在於部分函數需依賴分段線性近似與Lipschitz常數估計，導致初始化較慢，且優化過程仍對學習率敏感。整體而言，SLiR為神經網路驗證提供了更靈活且高效的工具。

**Research Gap:** 本文未明確說明SLiR在極大規模神經網路或更複雜架構（如Transformer）上的效能與可擴展性，且對於分段線性近似的段數選擇與Lipschitz常數估計的影響缺乏深入分析。優化過程對學習率敏感，可能影響實際部署的穩定性與效率。此外，SLiR主要針對單一輸入區間的鬆弛計算，對於動態輸入範圍或多維激活函數的處理未有探討。消融實驗雖展示部分改進效果，但對比其他最新優化策略仍有限，未涵蓋失敗案例或泛化能力評估。最後，全文未提供程式碼或實際部署細節，限制了工程落地的直接參考價值。

**Intro:** 本論文針對放射科影像中的視覺語言模型（VLM）訓練提出解決方案，特別聚焦於如何在缺乏人工空間標註的情況下，實現模型對2D CT與MRI影像的空間定位能力。作者建立了RefRad2D資料集，包含1.2百萬對德英雙語影像與文字對，並透過大型語言模型與自動分割技術自動生成空間標註。提出的RadGrounder模型能同時完成報告生成、視覺問答與空間定位任務，並在外部VQA基準測試中展現與專門醫療VLM相當的表現。此研究回應了放射影像中模型可解釋性與臨床信任的核心問題。

**Motivation:** 放射科VLM普遍缺乏對影像中具體區域的空間定位能力，導致模型輸出難以驗證，限制臨床應用與信任度。現有方法多依賴小規模或合成資料，且多集中於胸部X光，缺乏對CT與MRI大規模臨床資料的空間標註訓練。2D切片方法相較3D體積資料更具擴展性，但先前嘗試規模有限。如何在無需人工標註的前提下，利用大規模臨床資料訓練具空間定位能力的VLM，是推動臨床落地的關鍵挑戰。

**Method:** 作者提出RadGrounder，一個基於PaliGemma 2架構的多任務VLM，結合SigLIP視覺編碼器與Gemma語言解碼器，能自動生成報告、回答問題並進行空間定位。空間定位分為兩種策略：一是將邊界框座標作為離散詞彙由模型生成，無需額外分割頭或損失函數；二是輔以輕量分割頭進行像素級標註。資料集RefRad2D由臨床影像與報告組成，透過TotalSegmentator自動產生3D解剖分割，再切片並用LLM抽取關鍵詞與對應區域，形成空間標註。訓練時交替進行報告生成、VQA與空間定位任務，並提出LLMScore作為語言生成的臨床語義評估指標。

**Result:** RadGrounder在內部RefRad2D測試集及外部Slake與VQA-RAD基準上均展現競爭力表現。與多個基線模型比較，RadGrounder在報告生成、開放式VQA及空間定位任務均有優異成績，且加入空間標註訓練不影響語言生成品質。token-based邊界框定位策略在空間準確度（G-IoU）上優於分割頭，且無需額外模型複雜度。混合內部臨床資料與外部資料訓練提升了模型的泛化能力，尤其在開放式VQA指標上明顯優於僅微調下游資料的模型。

**Conclusion:** 本研究成功展示了利用大規模自動標註的臨床CT與MRI 2D切片資料，訓練具備空間定位能力的多任務放射科VLM的可行性。RadGrounder在多項任務中達到與專門醫療VLM相當的表現，且空間定位不損害語言生成品質，提升了模型的可解釋性與臨床信任度。提出的RefRad2D資料集與訓練策略為放射科VLM的擴展與應用提供了重要資源與方法論基礎。

**Research Gap:** 本論文的資料來源僅限於單一醫院，缺乏多中心資料驗證，限制了模型泛化能力的評估。空間標註僅涵蓋解剖結構，未包含病灶等病理性定位，降低臨床應用的直接價值。分割頭與邊界框定位兩種空間定位策略的比較因任務定義不同而難以直接量化，缺乏更全面的消融與比較分析。全文未明確說明模型在不同病理類型、影像品質變異或失敗案例的表現，亦未討論部署成本與實際臨床整合風險。未來工作應聚焦於多中心資料擴充、病灶級空間定位以及更細緻的性能與風險評估。

**Intro:** 本文針對離散時間的mean-field control問題，特別是在common noise的Wasserstein不確定性下，提出一種robust Q-learning演算法。mean-field control問題因涉及大量合作代理人，且代理人行為受群體分布影響，導致問題維度極高且難以求解。common noise的引入使得模型更貼近現實，但同時增加了分析與計算的複雜度。此外，common noise的真實分布通常未知，存在模型誤差與估計不確定性，這對控制策略的穩健性提出挑戰。本文旨在解決這些挑戰，設計一套能在common noise不確定性下學習最優策略的Q-learning方法。

**Motivation:** 在工程與產品實務中，mean-field control模型可用於交通系統、金融市場及疫情控制等場景，這些系統中存在共通隨機性(common noise)且其分布難以精確獲得。傳統方法難以處理高維度且缺乏對common noise分布不確定性的考量，導致策略在實際部署時表現不佳。robust Q-learning結合了分布式魯棒性與強化學習，能在有限資料與模型誤差下提升控制策略的可靠性，對於實際系統的穩健控制與風險管理具有重要意義。
