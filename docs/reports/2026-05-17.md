### 1. Sat3DGen: Comprehensive Street-Level 3D Scene Generation from Single Satellite Image

- 分數：8
- arXiv: [2605.14984v1](http://arxiv.org/abs/2605.14984v1)
- 發表日期：2026-05-14
- 分類：cs.CV, cs.AI
- 入選理由：+3 收錄於 Hugging Face Daily Papers; +3 提及頂級會議：iclr; +2 提及程式碼可用; LLM rank #1 / 9/10: Clear novelty in geometry-first 3D scene generation from satellite images with strong quantitative improvements and practical downstream applications; code and benchmark released.

**Intro:** 本論文聚焦於從單張衛星影像生成街景級別的3D場景，這是一項重要但具挑戰性的任務。現有方法在幾何精度與語義多樣性之間存在明顯取捨：幾何著色模型雖具高幾何忠實度，但多聚焦於建築且缺乏非建築語義；代理模型則透過前饋式影像到3D框架聯合學習幾何與紋理，內容豐富但幾何粗糙且不穩定。作者指出這些幾何失敗主要源於視角差異極大及監督稀疏不一致的衛星到街景資料特性。論文提出Sat3DGen，採用以幾何為核心的策略，結合新穎幾何約束與透視視角訓練，有效改善幾何錯誤，提升3D準確度與影像真實感。

**Motivation:** 街景3D場景在地圖製作、機器人導航、模擬與媒體創作等領域具實際應用價值，但地面捕捉成本高且覆蓋不均。衛星影像具廣域覆蓋、低成本與頻繁更新優勢，若能從單張衛星圖生成高品質街景3D，將大幅促進大規模長期應用。現有方法無法兼顧幾何精度與語義豐富，且視角差異與監督稀疏導致幾何不穩定，亟需新方法解決這些核心挑戰以提升實用性與可部署性。

**Method:** Sat3DGen基於前饋式影像到3D架構，採用凍結的DINO-v3 ViT編碼器將衛星影像轉為2D token網格，並透過VAE風格解碼器升採樣為三平面特徵。核心創新包括：重力基密度變化損失，鼓勵密度隨高度下降以抑制漂浮物與空洞；衛星視角單目相對深度先驗，利用Depth Anything v2產生的偽標籤約束屋頂深度，解決屋頂幾何模糊；空間token擴展邊界以穩定場景邊緣；以及透視視角訓練策略，結合全景與透視街景監督以增加視角覆蓋與光度一致性。渲染支援衛星視角、透視街景與全景，並可導出網格資產。

**Result:** 在新建的VIGOR-OOD測試集結合高解析DSM資料上，Sat3DGen在幾何RMSE由6.76米降至5.20米，顯著優於先前最佳Sat2Density++。影像真實感方面，FID從約40降至19，且無需額外影像品質模組。定量指標涵蓋FID、KID、DINO語義相似度、PSNR、SSIM及LPIPS，均顯示本方法優勢。消融實驗證明重力損失對真實感影響最大，深度先驗與空間token則對幾何精度貢獻顯著。質性結果展示更連貫的地面與邊界結構，屋頂與立面幾何更合理。下游應用包括語義地圖到3D合成、多鏡頭影片生成、大範圍網格生成及無監督DSM估計。

**Conclusion:** 本論文提出Sat3DGen，一種從單張衛星影像生成高品質街景3D場景的幾何優先方法。透過引入重力基密度變化損失、衛星視角深度先驗、空間token及透視視角訓練，顯著提升幾何精度與影像真實感。實驗結果在多項指標與視覺質量上均優於現有方法，並展示多樣化下游應用潛力。此方法有助於推動戶外3D場景生成研究與實務應用，如數位地球與場景模擬等。

**Research Gap:** 論文雖在VIGOR-OOD測試集展現優異表現，但僅限於特定城市與衛星影像解析度，泛化至其他地理區域或不同衛星資料尚未充分驗證。監督仍依賴街景全景影像，對無街景資料區域的適用性未明。重力損失與深度先驗雖有效，但對複雜非垂直結構或非典型建築形態的適應性未詳述。消融實驗未涵蓋所有可能組合，且訓練成本與部署效率未充分討論。失敗案例與模型在極端天氣、遮擋或動態場景下的魯棒性亦未呈現。未來研究可針對這些限制進行更全面評估與優化。
