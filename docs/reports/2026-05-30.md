### 1. When Should Models Change Their Minds? Contextual Belief Management in Large Language Models

- 分數：7
- arXiv: [2605.30219v1](http://arxiv.org/abs/2605.30219v1)
- 發表日期：2026-05-28
- 分類：cs.AI, cs.CL, cs.LG
- 入選理由：+2 提及頂級機構：mit; +3 收錄於 Hugging Face Daily Papers; +2 提及程式碼可用; LLM rank #1 / 9/10: Introduces a novel, measurable framework (Contextual Belief Management) for LLM state updates with strong empirical evidence and a new benchmark; addresses a fundamental LLM limitation with clear practical and research i

**Intro:** 本論文探討大型語言模型在長期互動中如何管理累積資訊，特別是何時更新、保持或忽略其信念狀態。這個問題被定義為Contextual Belief Management（CBM），即模型需維持與正式證據一致的信念狀態，同時隔離任務無關的噪音。為了可量化CBM，作者提出BeliefTrack基準，涵蓋Rule Discovery與Circuit Diagnosis兩個封閉世界環境，利用有限信念空間與符號驗證器實現精確的逐輪評估。實驗發現多數現有大型模型在CBM上表現不佳，且僅用提示提升有限。

**Motivation:** 在實際工程與產品應用中，語言模型常面臨長期多輪互動，必須正確管理信念狀態以避免錯誤累積或被無關噪音干擾。現有研究多聚焦於模型吸收上下文資訊，卻忽略何時應更新信念或忽略無關訊息。CBM的提出有助於精確診斷模型在信念保持、更新與隔離噪音上的失誤，對提升模型在複雜任務中穩定性與可靠性具實務意義。BeliefTrack提供可自動評估的基準，有助於工程師針對信念管理問題進行系統化優化。

**Method:** CBM被形式化為模型在多輪互動中維持與正式證據一致的信念狀態。BeliefTrack設計兩個封閉世界任務：Rule Discovery和Circuit Diagnosis，定義有限信念空間與符號驗證邏輯，能精確判斷模型預測信念狀態與正確狀態的吻合度。作者定義三種失敗模式：Failed Stay（未保持穩定信念）、Failed Update（未正確更新信念）、Failed Isolation（未隔離無關噪音）。提升方法包括無需訓練的Belief-Tracking Prompt（BT-Prompt）與基於符號驗證器回饋的強化學習（RL）策略，後者利用Jaccard相似度作為信念狀態獎勵，優化模型在特定輪次的信念預測。

**Result:** 實驗涵蓋Qwen2.5-7B-Instruct與Qwen3.5-9B兩種模型，評估三種失敗率指標及跨任務泛化能力。結果顯示，原始模型在CBM上失敗率極高（多達97%以上），BT-Prompt提升有限且不穩定，有時甚至惡化表現。強化學習方法顯著降低失敗率，平均減少超過70%，且在未見任務上仍有良好泛化能力。此外，RL訓練後模型在面對時間延遲證據與多種噪音干擾時展現更強的魯棒性。機制分析透過提示探查與表示層調控，揭示RL能改善信念漂移、回溯失敗與上下文劫持等問題。

**Conclusion:** 本研究首次系統化提出Contextual Belief Management問題，並透過BeliefTrack基準精確量化大型語言模型在信念管理上的缺陷。實驗證明現有模型在CBM上表現不佳，且簡單提示不足以解決，反而透過符號驗證器指導的強化學習能有效提升信念狀態的準確性與穩定性，且具跨任務泛化能力。機制分析進一步指出CBM失敗與模型內部表示相關，代表未來可從表示層面持續優化。限制在於BeliefTrack僅涵蓋兩個合成封閉世界任務，未涵蓋更開放或複雜的信念修正場景，且真實世界中證據與噪音界線不明確，模型過度過濾可能影響對用戶反饋的敏感度。

**Research Gap:** 本論文雖在封閉世界環境中精確評估CBM，但缺乏對開放式、多樣化任務的驗證，無法保證方法在真實應用中同樣有效。強化學習訓練未包含噪音干擾樣本，雖有泛化效果，但對更複雜噪音類型的魯棒性仍未充分測試。BT-Prompt效果不穩定，顯示提示設計仍有改進空間。論文未明確說明強化學習訓練的計算成本與部署風險，對實際工程落地的影響尚不清楚。此外，對失敗案例的細節分析有限，缺乏對模型在多輪長期互動中可能累積錯誤的深入探討。未來需擴展至更貼近真實場景的信念管理問題，並評估方法在多模態或跨領域任務中的適用性。

### 2. Physics Is All You Need? A Case Study in Physicist-Supervised AI Development of Scientific Software

- 分數：7
- arXiv: [2605.30353v1](http://arxiv.org/abs/2605.30353v1)
- 發表日期：2026-05-28
- 分類：cs.AI, astro-ph.CO, cs.HC, cs.SE
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：acl, icml; +2 提及程式碼可用; LLM rank #2 / 8/10: Provides a detailed, credible case study on physicist-supervised AI for scientific software development, revealing important insights on supervision design beyond model scaling; practical and novel with transparent evide

**Intro:** 本論文呈現一個物理學家監督AI編碼代理人開發科學軟體的單案例研究，聚焦於建構一個基於JAX的可微分一環擾動理論模組CLAX-PT。該模組用於預測星系聚類，並與既有的C語言參考實作CLASS-PT進行精度驗證。研究記錄了15次監督事件，分析人類與AI代理人各自解決問題的能力與限制。此案例反映了科學軟體開發中正確性不僅是測試通過，更需符合物理理論的嚴格要求。

**Motivation:** 科學軟體的正確性不同於一般軟體，必須符合物理定律而非僅通過編譯或測試。現有AI編碼代理多聚焦於標準化測試或完全自主系統，缺乏對科學軟體中物理正確性與人機協作的深入探討。此研究旨在揭示監督協議如何影響AI生成科學軟體的可信度，並探討AI在物理理論驅動軟體開發中無法自發解決的關鍵問題，對工程與產品落地具有指導意義。

**Method:** 研究中物理學家監督使用Claude Code、Sonnet與Opus模型的AI代理人，透過57個工作階段、12個工作日，依據預先設計的監督協議進行開發。該協議包含以CLASS-PT作為oracle測試基準、共享變更日誌防止重複錯誤探索、限制輸出訊息以維持上下文清晰，以及並行多代理人會話探索多種假設。核心方法是讓AI在oracle測試指引下自動修正錯誤，並由物理學家介入進行架構重設與物理合理性判斷，確保軟體不僅數值正確且物理意義正確。

**Result:** AI代理人自主解決了15個監督事件中的10個，包括慣例錯誤、算法實作錯誤及數值係數調整。兩個問題因物理學家發現數值量級差異而加速解決。三個關鍵問題無法由AI解決，因為AI無法辨識錯誤架構或偽裝成正確的校準因子（fudge factor）。經物理學家介入，進行架構重設（引入非各向同性BAO阻尼）、拒絕無物理依據的校準因子，最終使所有九個輸出功率譜達到約1%以內的誤差。研究強調監督協議設計對結果可信度的決定性影響。

**Conclusion:** 本案例顯示AI代理人作為工具能有效執行科學軟體的實作與除錯，但缺乏自發質疑與重構解決方案架構的能力。監督協議中包含oracle測試、多參數點驗證、共享日誌及明確拒絕無物理依據修補的規則，是確保軟體物理正確性的關鍵。未來提升可靠性的方向不僅是提升模型能力，更需設計能促使代理人提出架構替代方案及具備解釋性判斷的監督流程。此研究對AI輔助科學軟體開發的工程實踐與責任歸屬提供重要啟示。

**Research Gap:** 本研究為單一案例，僅涵蓋一種代理人架構、一個物理領域及一位監督者，缺乏多樣性與獨立評審。代理人無法自發提出架構重設，且無法辨識無物理依據的校準因子，顯示解釋性判斷能力不足。實驗中未明確量化推理成本與效能，且未進行對比性消融實驗以區分檢索輔助與物理概念注入的影響。此外，監督協議依賴人類專家介入，尚未探討如何自動化或半自動化此過程。未來研究需擴展至多代理人、多領域、多監督者，並設計能促使代理人具備架構選擇與解釋能力的機制。

### 3. How LoRA Remembers? A Parametric Memory Law for LLM Finetuning

- 分數：7
- arXiv: [2605.30260v1](http://arxiv.org/abs/2605.30260v1)
- 發表日期：2026-05-28
- 分類：cs.CL, cs.AI, cs.CV, cs.LG
- 入選理由：+2 提及頂級機構：mit; +3 收錄於 Hugging Face Daily Papers; +2 提及程式碼可用; LLM rank #3 / 8/10: Quantitatively characterizes LoRA finetuning memory capacity with a novel Parametric Memory Law and proposes an effective optimization strategy; strong theoretical and empirical contributions with practical relevance.

**Intro:** 大型語言模型（LLMs）在實際應用中需持續學習與更新知識，以應對動態環境。然而，現有方法多依賴非參數式記憶，受限於固定上下文窗口與計算成本。LoRA作為一種低秩適配技術，廣泛用於參數高效微調，並被視為一種模組化的參數記憶單元，但其精確記憶容量與動態機制尚未被系統性量化。本文以LoRA作為可控記憶容量探針，提出參數記憶定律，並揭示token層級的確定性相變現象，進一步設計MemFT優化策略提升記憶效率與準確度。

**Motivation:** 在實務中，LLMs需不斷吸收新知識以維持效能，然而非參數式方法如檢索增強生成受限於上下文長度及注意力稀釋，難以長期存儲大量資訊。參數式記憶則可將知識永久編碼於模型參數中，避免推理時的檢索開銷，但目前對其容量極限及記憶動態缺乏量化理解。理解這些機制對工程師設計高效微調方案、提升模型記憶精度及部署穩定性具有重要意義，尤其在需要精確回憶的場景如法律文本、個人憑證等。

**Method:** 本文將LoRA視為潛在空間中的參數記憶探針，通過調節LoRA秩（rank）和序列長度，系統性掃描記憶容量。提出參數記憶定律，描述loss減少量與有效參數數量及序列長度之間的冪律關係。進一步分析token層級的預測概率，發現當token預測概率超過0.5時，該token可被確定性地記憶，低於此閾值則易引發自回歸錯誤級聯。基於此，設計MemFT方法，透過閾值引導的動態權重分配，將訓練資源集中於未達閾值的困難token，提高記憶效率與準確率。

**Result:** 實驗在Qwen3-8B-IT與Llama3.1-8B-IT兩個模型上進行，涵蓋長上下文隨機token替換壓力測試與短文本PhoneBook記憶任務。結果顯示參數記憶定律在多種語義密度與序列長度下均有極高擬合度（R2>0.98）。token層級分析揭示loss與準確率不一致現象，並定位記憶失敗多集中於少數困難token。MemFT在不同參數容量下均優於標準微調，長上下文任務中低秩時MemFT-SW表現最佳，高秩時MemFT-OT達到完美記憶。PhoneBook任務中MemFT-SW持續領先，且在線性規則學習任務中亦提升泛化能力。

**Conclusion:** 本文首次系統量化LoRA作為參數記憶單元的容量與動態機制，提出參數記憶定律揭示loss減少與參數及序列長度的冪律關係，並發現token層級存在確定性相變，定義了記憶成功的概率閾值。基於此，MemFT方法有效提升了記憶精度與參數利用效率。該研究對工程實踐中設計高效微調策略及理解參數記憶機制具有指導意義，但仍限於8B規模模型與貪婪解碼，未涵蓋更大模型與隨機解碼策略。

**Research Gap:** 本研究僅驗證於8B規模模型，尚未探討參數記憶定律在更大或更小模型上的泛化性。確定性相變分析僅針對貪婪解碼，對隨機解碼（如nucleus sampling）下的穩健性未明確說明。MemFT雖提升記憶與泛化，但對於開放式推理等更廣泛能力的影響未充分評估。實驗中缺乏對成本、訓練時間及部署風險的量化分析，且未展示失敗案例或極端場景下的表現。未來工作可補充多樣化任務、模型規模及解碼策略的全面驗證。

### 4. EASE Configuration Facilitates A Reproducible Science of LLM Social Simulations

- 分數：7
- arXiv: [2605.30258v1](http://arxiv.org/abs/2605.30258v1)
- 發表日期：2026-05-28
- 分類：cs.MA
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：neurips; +2 提及程式碼可用; LLM rank #4 / 7/10: Proposes a modular, reproducible framework (EASE) for LLM social simulations with open-source implementation and case studies; advances rigor and reproducibility in an important emerging area.

**Intro:** 本論文聚焦於大型語言模型（LLM）在社會互動模擬中的應用，指出現有模擬器多為臨時且單一架構，缺乏標準化，導致研究難以重現且評估困難。作者提出EASE框架，將模擬系統拆解為環境、代理人、模擬引擎與評估指標四大模組，強調模組化設計有助於明確區分各組件對結果的影響。論文同時提供開源工具SiliSocS，支持基於EASE的可配置且可重現的社會模擬研究。透過三個案例研究，展示此架構在評估、深入探討及擴展現有研究上的實用性與優勢。

**Motivation:** 隨著LLM在多代理社會模擬中被廣泛採用，現有系統多為單一且緊密耦合的設計，難以釐清模擬結果的成因，阻礙科學推論與重現性。缺乏統一架構與評估標準，使得研究成果難以比較與驗證，且容易因設計偏差產生誤導性結論。對工程與產品而言，這種不透明與不可重現的模擬限制了模擬結果的可信度與應用價值。EASE框架旨在解決這些問題，促進模擬設計的透明化與模組化，提升研究的可驗證性與工程實踐的可控性。

**Method:** EASE框架將社會模擬拆解為四個核心組件：環境（定義代理人互動的社會空間及其介面）、代理人（結合LLM模型、指令、角色設定、記憶結構與認知架構）、模擬引擎（控制時間步長、代理人行動順序與活躍度）及評估指標（包含參考場景、模擬中探針與後設分析）。此外，作者提出一套結構化的實驗研究流程，將研究問題轉化為具體假設，明確定義變量與評估標準，並透過SiliSocS開源工具實現配置化與可重現的模擬執行。此方法支持逐步假設驗證與組件間的因果分析。

**Result:** 論文透過三個案例展示EASE的應用價值。第一個案例探討風格多樣性，發現模型強度顯著影響文本多樣性，而豐富的角色設定主要影響立場多樣性，且提示工程可彌補模型弱勢。第二個案例分析社交媒體模擬中推薦算法對用戶參與度的影響，揭示行動限制與代理人提示會調節推薦效果，且模型規模亦影響參與行為。第三個案例重現並擴展既有回音室研究，透過模組化設計明確分離環境、代理人與引擎假設，展示不同組件對極化現象的貢獻，強化了研究的可重現性與解釋力。

**Conclusion:** 本論文提出的EASE框架及其實現SiliSocS工具，為LLM社會模擬提供了一套結構化、模組化且可重現的研究方法。透過明確分離模擬組件與評估指標，促進了因果推論與假設驅動的實驗設計，提升了模擬結果的透明度與可信度。三個案例研究展示了該框架在多樣性、參與度及極化等社會現象模擬中的實用性與靈活性。論文強調此架構有助於推動LLM社會模擬從觀察性模仿向嚴謹科學推進，並為社會科學與AI安全等領域提供支持。

**Research Gap:** 儘管EASE框架促進了模組化與可重現性，但論文未充分解決LLM社會模擬的真實世界驗證問題，缺乏與實際社會數據的對比與外部驗證。實驗設計雖支持假設驅動探索，但面對龐大設計空間仍存在計算成本高昂與覆蓋不足的挑戰。部分模組如評估指標的選擇依賴用戶，可能導致過度擬合或偏差，且對失敗案例與泛化能力的討論有限。最後，案例研究多為示範性質，未提供足夠的跨場景或跨模型的廣泛比較，限制了結論的普適性與應用範圍。

### 5. Demystifying Data Organization for Enhanced LLM Training

- 分數：7
- arXiv: [2605.30334v1](http://arxiv.org/abs/2605.30334v1)
- 發表日期：2026-05-28
- 分類：cs.AI, cs.CL
- 入選理由：+2 提及頂級機構：microsoft; +3 提及頂級會議：acl; +2 提及程式碼可用; LLM rank #5 / 7/10: Systematic study of data organization effects on LLM training efficiency with novel methods and extensive experiments; practical value for improving LLM training stability and performance.

**Intro:** 本論文聚焦於大型語言模型（LLM）訓練中資料組織策略的重要性，指出目前多數研究著重於資料選擇，卻忽略了資料呈現順序對訓練效率與效果的影響。由於LLM通常只訓練少數幾個epoch，資料的時間序列安排成為關鍵因素。作者利用先前計算的樣本層級分數，提出系統化的資料組織指導原則，並設計兩種新穎的資料排序方法STR與SAW，旨在提升訓練穩定性與模型表現。

**Motivation:** 在LLM訓練中，資料的質量與選擇固然重要，但資料的呈現順序對模型學習軌跡有顯著影響。由於訓練通常只遍歷資料一至數次，若資料排序不當，模型可能遺忘早期學習的基礎知識或遭遇優化震盪，導致性能下降。現有方法如課程學習（Curriculum Learning）未能全面考慮資料組織的多面向需求，缺乏系統性指導，且增加訓練成本。故本研究旨在提出低成本且有效的資料組織策略，促進LLM訓練的穩定與效率。

**Method:** 作者首先定義資料組織問題，將資料排序視為基於預先計算的樣本分數的排列問題，並提出四項核心指導原則：邊界銳化（Boundary Sharpening）、循環排程（Cyclic Scheduling）、課程連續性（Curriculum Continuity）與局部多樣性（Local Diversity）。基於此，設計了四個對應演算法SEG、FO、ZIG與JIT，並將其組合成兩種跨指導策略STR與SAW。STR結合邊界銳化、循環排程與局部多樣性，SAW則進一步加入課程連續性以平滑資料屬性過渡。整體方法重用已有分數，無需額外計算成本。

**Result:** 實驗涵蓋多種模型規模（160M至1.7B參數）與資料集（FineWeb-Edu、QuRatedPajama、DeepMath-103K、OpenCodeInstruct），在預訓練與監督微調階段均驗證指導原則有效性。結果顯示，結尾使用高分資料顯著提升性能，循環排程減少遺忘現象，課程連續性降低優化震盪，局部多樣性提升模型魯棒性。STR與SAW策略在多項下游任務中均超越隨機排序及傳統課程學習基線，且在大規模資料與模型上持續展現優勢。透過Chinchilla Scaling Law推估，資料組織效益在更大模型與訓練規模下仍具延展性。

**Conclusion:** 本研究首次系統性探討資料組織對LLM訓練的影響，提出四項具體指導原則並設計兩種有效排序策略STR與SAW。實驗結果證明這些策略能提升訓練穩定性與模型性能，且適用於不同模型規模與資料集。方法重用既有分數，計算成本低，為LLM訓練提供新的優化視角。限制方面，策略效果依賴於預先計算分數的品質與相關性，且全文未明確說明在更複雜任務或多模態場景下的適用性與潛在風險。

**Research Gap:** 本論文雖提出系統化資料組織指導，但仍存在多項不足。首先，策略依賴預先計算的樣本分數，若分數不準確或不具代表性，可能影響排序效果，全文未深入探討分數品質對結果的敏感度。其次，實驗主要聚焦於語言模型與特定資料集，缺乏跨領域、多模態或更複雜任務的驗證。第三，對於策略在實際大規模分布式訓練環境中的部署成本與效益權衡未充分說明。第四，消融實驗雖涵蓋多種指導原則，但對不同參數設定的穩定性分析有限。最後，未提供失敗案例或策略可能導致性能下降的情境，限制了對方法泛化性的全面評估。
