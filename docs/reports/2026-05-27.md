### 2. L2IR: Revealing Latent Intent in Graph Fraud Detection

- 分數：5
- arXiv: [2605.26040v1](http://arxiv.org/abs/2605.26040v1)
- 發表日期：2026-05-25
- 分類：cs.AI
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：acl; LLM rank #2 / 8/10: Introduces a novel LLM-driven framework to reveal latent intent in graph fraud detection, addressing a real-world challenge of camouflage in fraud signals with credible evaluation and significant performance gains.

**Intro:** 圖形詐騙偵測長期依賴圖神經網絡（GNN）來聚合關聯數據資訊，但詐騙者常透過與正常用戶建立大量連結進行偽裝，導致詐騙訊號在鄰居聚合過程中被稀釋，降低偵測準確度。近年來大型語言模型（LLM）被用於提供豐富語義線索，但對於可疑連結背後的潛在意圖探討仍不足。此外，標註詐騙樣本稀缺使得模型難以在重度偽裝下保持魯棒。本文提出L2IR框架，利用LLM揭露用戶行為及連結的潛在意圖，並結合自適應自訓練以提升有限標註下的偵測能力。

**Motivation:** 現有GNN方法主要依賴鄰居聚合，無法有效辨識詐騙者偽裝成正常用戶的行為，導致詐騙訊號被稀釋且誤判率升高。雖有研究嘗試結合LLM提供語義輔助，但多半只生成全局語義描述，未深入推理連結背後的真實意圖，無法充分利用LLM的推理能力。加上詐騙標註稀少，模型訓練面臨監督不足的挑戰。這些問題直接影響工程與產品在實際場景中對隱蔽詐騙的偵測效果與穩定性。

**Method:** L2IR包含三大模組：行為意圖剖析、連結意圖推理與自適應自訓練。行為意圖剖析透過LLM分析用戶歷史行為文本，結合動態檢索的標註示例，生成節點層級的行為意圖表示。連結意圖推理先用初步GNN評分節點，識別矛盾邊後，利用LLM交叉審核連結雙端的互動歷史，推斷連結背後的真實意圖並編碼為邊特徵。最後，自適應自訓練透過多輪迭代，利用高置信度偽標籤擴充訓練集，並在每輪重新初始化GNN以避免錯誤累積，提升模型在標註稀缺下的魯棒性。

**Result:** 在Amazon與Yelp兩個真實世界資料集上，L2IR結合CARE-GNN作為基礎模型，在不同訓練比例（10%至40%）下均優於九個先進基線，尤其在低標註比例時提升更明顯。作為插件整合至多種GNN模型，L2IR在標註減半且40%訓練比例條件下，平均提升AUROC約2%至3%，AUPRC提升達5%以上，最高提升達8.27%。消融實驗證實行為意圖剖析、連結意圖推理與自適應自訓練均對性能有顯著貢獻。參數敏感度分析顯示模型對自訓練置信閾值及類別平衡具一定魯棒性。效能方面，L2IR訓練時間與主流GNN相當，LLM推理作為離線預處理，未增加線上訓練負擔。

**Conclusion:** 本文提出的L2IR框架透過LLM推理用戶行為與連結意圖，有效揭露隱蔽詐騙，並結合自適應自訓練緩解標註稀缺問題。實驗結果證明L2IR在多種資料集與GNN基線上均有穩定且顯著的提升，且不增加線上訓練成本。此方法對工程與產品實際部署具參考價值，尤其在面對高偽裝詐騙與有限標註時。全文未明確說明在更大規模圖或其他類型詐騙場景的泛化能力及失敗案例，且LLM推理成本與延遲在實際系統中仍需評估。

**Research Gap:** 本文雖展示L2IR在兩個資料集的優異表現，但缺乏對更大規模或多樣化圖數據的實驗驗證，泛化能力尚不明確。自適應自訓練的偽標籤生成策略依賴置信閾值設定，可能在標註極度稀缺或異常分布下表現不穩。LLM推理雖離線執行，但對資源需求與延遲影響未深入探討，對實時系統部署存在潛在風險。消融實驗未涵蓋不同LLM模型架構或大小的影響，且缺少對失敗案例的分析，限制對方法穩健性的全面理解。未明確說明對多種詐騙類型或動態圖的適應性，未涉及對抗攻擊或惡意操控的防禦能力。

### 3. Neural Scalable Symbolic Search Framework for Complex Logical Queries with Multiple Free Variables

- 分數：5
- arXiv: [2605.25985v1](http://arxiv.org/abs/2605.25985v1)
- 發表日期：2026-05-25
- 分類：cs.AI
- 入選理由：+3 提及頂級會議：kdd; +2 提及程式碼可用; LLM rank #3 / 8/10: Presents a scalable neural symbolic search framework for complex multi-variable logical queries on knowledge graphs, with a new benchmark and strong empirical improvements, advancing fundamental KG reasoning tasks.

**Intro:** 本論文聚焦於複雜邏輯查詢中含有多個自由變數的存在性一階邏輯查詢（EFO_k queries），這類查詢在知識圖譜（KG）推理中極具挑戰性。隨著自由變數數量k增加，答案空間呈指數成長，導致傳統方法難以有效處理。現有方法多依賴對單一變數的邊際排名，無法準確反映多變數聯合排名，限制了實際應用的準確性與效率。

**Motivation:** 多自由變數查詢在金融詐騙偵測、社交網絡分析等場景中有重要應用價值，但聯合排名的計算複雜度極高，現有基準與方法多以邊際排名替代，造成結果不夠精確。為了提升推理的準確性與可擴展性，亟需一種能在有限計算預算下，直接逼近聯合排名的有效框架，促進複雜查詢在工程與產品中的落地。

**Method:** 作者提出Neural Scalable Symbolic Search（NS3）框架，核心在於通過兩大轉換：邊際化（marginalization）與合併（merge），將EFO_k查詢逐步簡化為EFO_1查詢。邊際化用於獲取必要的候選集合以進行安全剪枝，合併則將多個自由變數合併成超節點，並在動態預算B控制下構建縮減的聯合域。此過程透過預算分配策略，逐步合併變數，避免指數級搜索空間，並利用神經符號搜索推理聯合排名。實驗中，NS3基於NLISA實現，並在三個標準KG上評估。

**Result:** 在FB15k-237等三個知識圖譜上的EFO_k-CQA基準測試中，NS3在邊際排名與聯合排名指標均超越多個代表性基線模型，包括BetaE、LogicE、ConE、CQD、FIT等。特別是在聯合排名上，NS3提升明顯，表明其有效逼近聯合解空間。作者另構建擴展至k=3的可擴展基準，涵蓋多種自由變數依賴結構，NS3在此基準上同樣展現出顯著優勢，且隨預算增加性能提升趨於飽和，顯示預算分配策略有效平衡效率與準確度。消融實驗證實逐步預算合併策略優於一次性合併。

**Conclusion:** 本論文首次提出一種可擴展且具預算控制的神經符號搜索框架NS3，實現了對多自由變數存在性一階邏輯查詢的直接聯合排名推理。方法有效縮減搜索空間，兼顧推理準確性與計算效率，並在多個標準與新構建基準上取得優異表現。NS3為複雜邏輯查詢的實際應用提供了可行方案，尤其適合需要多變數聯合推理的場景。

**Research Gap:** 論文雖展示NS3在k≤3時的有效性，但對更大自由變數數量的擴展性與性能尚未充分驗證，全文未明確說明在k顯著增大時的計算成本與精度表現。此外，對於不同類型知識圖譜（如動態、多模態KG）及更複雜邏輯結構的適用性未展開實驗。消融分析主要聚焦預算策略，對於合併順序、剪枝準確性等細節影響缺乏深入探討。實際部署時的資源消耗、延遲及可解釋性等工程層面挑戰亦未充分討論。最後，雖有程式碼釋出，但對於大規模工業場景的適配與優化策略仍待研究。

### 4. SemBridge: Language Transfer in Sparse Encoders via Multilingual Semantic Bridges

- 分數：5
- arXiv: [2605.26002v1](http://arxiv.org/abs/2605.26002v1)
- 發表日期：2026-05-25
- 分類：cs.IR
- 入選理由：+2 提及頂級機構：mit; +3 收錄於 Hugging Face Daily Papers; LLM rank #4 / 8/10: Proposes SemBridge, a practical and effective method for cross-lingual transfer in sparse retrieval encoders, validated on multiple languages and architectures with consistent improvements and clear practical value.

**Intro:** 稀疏編碼器因能在詞彙空間中表示詞語重要性，提供高精度檢索，但多數模型以英語為中心，導致非英語語言的遷移困難。這種結構性限制使得直接微調英語稀疏編碼器以適應目標語言效果有限。本文提出SemBridge，一種利用多語言密集嵌入作為語義橋樑，實現跨語言詞彙對齊的嵌入初始化方法，旨在提升稀疏編碼器的語言遷移能力。

**Motivation:** 隨著全球資訊需求增加，非英語語言的檢索系統需求日益迫切，但現有稀疏編碼器詞彙多為英語，缺乏目標語言詞彙維度，造成語義表達不足。直接微調無法克服詞彙空間結構限制，導致檢索性能受限。工程與產品層面，若無有效遷移方法，需重新訓練語言專屬模型，成本高昂且資料需求大。SemBridge透過語義橋樑精準初始化目標語言詞嵌入，提升模型遷移效率與效果，對實際部署多語言檢索系統具重要意義。

**Method:** SemBridge方法包含三大步驟：首先，對於源語言與目標語言詞彙中重疊的詞，直接複製其嵌入以保留通用語義。其次，利用多語言密集嵌入模型將源詞與目標詞映射至共享語義空間，計算目標詞與所有源詞的相似度向量。最後，透過Entmax稀疏化轉換，選擇少數語義相關的源詞作為權重，將目標詞嵌入初始化為這些核心同義詞的線性組合。此過程保持嵌入維度不變，無需修改模型架構，並有效過濾語義噪聲。

**Result:** 在五種語言（阿拉伯語、中文、印地語、韓語、俄語）及四種稀疏編碼器架構上，SemBridge在零樣本設定下即展現優異檢索性能，平均nDCG@10顯著優於隨機初始化及其他語言遷移基線方法。微調後，SemBridge持續保持性能領先，且收斂速度更快，訓練初期損失明顯較低。稀疏權重調整實驗顯示適度稀疏化（Entmax α=3或4）能有效提升語義對齊精度。效率分析指出SemBridge在保持高性能的同時，計算成本低於部分基線。質性分析證實SemBridge能準確對齊目標詞與核心同義詞，避免無關詞彙干擾。

**Conclusion:** 本文提出的SemBridge方法成功解決了英語中心稀疏編碼器在非英語語言遷移中的結構性瓶頸，通過多語言語義橋樑與稀疏權重機制，實現精確且高效的目標詞嵌入初始化。實驗結果證明其在多語言、多模型環境下均具優越的零樣本與微調後檢索表現，並加速訓練收斂。SemBridge為工程實踐中部署多語言稀疏檢索系統提供了可行且高效的解決方案。限制方面，方法對Entmax稀疏參數敏感，且僅在五種語言驗證，對其他語言類型與資源稀缺語言的泛化尚未充分評估。

**Research Gap:** 本文雖在五種語言與四種稀疏編碼器上驗證SemBridge，但未涵蓋更多語言類型，尤其是資源極度匱乏或語系結構差異大的語言，泛化能力仍待驗證。Entmax稀疏參數α的選擇對性能影響顯著，但缺乏自適應或語言感知的調整策略，可能限制實際應用靈活性。實驗中未明確討論在極端低資源或無標訓練資料情境下的表現與失敗案例。此外，全文未詳細說明在大規模工業環境中部署的計算成本與延遲影響，對模型更新與維護的工程挑戰亦未涉及。未來研究可聚焦於跨語言自適應稀疏化策略、更多語言覆蓋及實際系統整合測試。

**Intro:** 異常檢測在風險敏感領域如詐欺防範與醫療診斷中扮演重要角色，但實務中常面臨標註不完整、粗糙或含噪聲的弱監督問題。弱監督異常檢測（WSAD）主要分為不完整、粗糙及不準確三種監督缺陷，但現有研究多半孤立發展，缺乏統一評估框架。本文提出WSADBench，首個跨多種弱監督場景的統一基準，涵蓋36種算法與4種資料型態，並透過超過70萬次實驗揭示不同方法的性能邊界與相互關係。

**Motivation:** 現有WSAD研究方向分散且缺乏可比較性，導致無法判斷不同監督缺陷是否為獨立問題，或是否可互相轉換。此外，專門設計的WSAD方法是否必要，與通用基礎模型相比的優劣尚未明確。不同研究使用不一致的特徵抽取器與評估協議，阻礙跨方法的公平比較。這些問題限制了WSAD技術在工程與產品中的落地與推廣，因此需要一個標準化、全面且跨模態的評測平台。

**Method:** WSADBench建立統一評測框架，涵蓋不完整、粗糙、不準確三種弱監督場景，並測試方法間的跨場景轉移能力。收錄61個涵蓋表格、影像、文本與影片的資料集，並統一使用預訓練特徵與標準化標註定義。評測36種方法，包括專門WSAD算法、通用分類器與表格基礎模型。實驗設計嚴格控制標註比例、標註粒度與標註品質，並採用AUC-ROC與AUC-PR指標，確保結果可重現且具統計顯著性。

**Result:** 實驗發現專門WSAD方法僅在極度標註稀缺時優於通用監督方法，隨著標註增加，表格基礎模型如LimiX與TabPFN表現超越專門方法。影像與文本高維度特徵降低基礎模型優勢。未標註資料的效用依賴標註量，標註極少或極多時效益有限，中等標註量時提升最大。模型對標註噪聲呈現不對稱敏感，正常標註錯誤影響較大。WSAD方法在異常類型分布外（OOD）泛化能力有限，容易過度擬合已知異常。跨監督類型轉移實驗顯示，專門方法受限於領域偏好，通用方法具較好適應性。

**Conclusion:** WSADBench提供首個跨監督缺陷與多模態的統一評測平台，揭示專門WSAD方法在極端稀缺標註下的價值，但隨標註增加通用基礎模型更具競爭力。研究強調弱監督場景間存在內在關聯，且模型對標註噪聲與OOD異常敏感。此基準促進WSAD領域的系統性理解與公平比較，並為未來結合基礎模型與強化泛化能力的研究指明方向。全文未明確說明實際部署成本與效能平衡。

**Research Gap:** 本文雖涵蓋多模態與多種弱監督場景，但對高維度影像與文本特徵下基礎模型劣勢的深入分析不足，缺乏對模型在真實產業場景中部署成本與延遲的評估。OOD泛化實驗多基於PCA距離劃分，未涵蓋更複雜的異常類型變化。噪聲模型僅模擬標註翻轉，未探討其他噪聲類型如系統性偏差。跨監督轉移實驗主要限於部分方法與資料集，缺少更廣泛的消融與比較。未提供對失敗案例的詳細分析，限制工程師在實務中選擇方法的參考價值。
