### 1. The Double Dilemma in Multi-Task Radiology Report Generation: A Gradient Dynamics Analysis and Solution

- 分數：7
- arXiv: [2605.22635v1](http://arxiv.org/abs/2605.22635v1)
- 發表日期：2026-05-21
- 分類：cs.LG, cs.CL, cs.CV
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：icml; +2 提及程式碼可用; LLM rank #1 / 9/10: Strong novelty in multi-task radiology report generation optimizer with solid theoretical analysis and consistent empirical improvements across multiple methods and datasets; code available and accepted at ICML.

**Intro:** 本論文聚焦於多任務學習在自動放射科報告生成（RRG）中的優化問題。現有方法多著重於架構設計，但普遍採用粗糙的線性加權策略，無法有效平衡臨床監督的嚴格約束與報告生成的語言流暢性需求。作者從梯度動力學角度出發，利用隨機微分方程（SDE）框架，揭示線性加權策略導致的優化失效機制，稱之為「雙重困境」：漂移項偏差與擴散項衰減。基於此，提出一種通用的優化器CAME-Grad，旨在解決此困境並提升多任務RRG的性能。

**Motivation:** 多任務學習在RRG中引入多種臨床約束以提升診斷準確性，但線性加權的優化策略無法有效處理任務間的梯度衝突。這種衝突導致優化方向偏離帕累托最優軌跡，且梯度幅度衰減，限制模型逃離局部最優解的能力。這對工程實踐和產品部署造成挑戰，因為生成報告可能忽略關鍵病灶或語言表達不佳，影響臨床可信度。故深入理解並改進多任務優化策略對於提升RRG系統的臨床效用至關重要。

**Method:** 作者首先從理論上分析多任務RRG中梯度的幾何衝突與動力學，定義任務梯度間的負餘弦相似度為衝突指標，並利用SDE模型將優化過程分解為漂移項（決定收斂方向）與擴散項（提供探索動能）。提出的CAME-Grad優化器包含三個階段：一是衝突迴避方向校正，透過凸優化在信賴域內調整梯度方向以減少衝突；二是幅度增強能量注入，恢復並放大梯度幅度以避免擴散項衰減；三是自適應梯度融合，平衡理論最優方向與任務特定偏差，保留細粒度語義信息。該方法可作為通用插件優化器，無需修改模型架構。

**Result:** 在MIMIC-CXR和IU X-Ray兩大公開數據集上，作者將CAME-Grad應用於八種代表性RRG模型，實驗結果顯示臨床效能指標平均提升2.3%和1.9%。與多種多任務優化器比較，CAME-Grad在保持生成語言質量的同時，顯著提升診斷準確率，尤其在處理梯度衝突和逃離局部最優方面表現優異。消融實驗證實三個階段均對性能提升有貢獻。質性分析展示CAME-Grad能更準確捕捉罕見且關鍵的病理特徵，減少漏診風險。

**Conclusion:** 本研究從梯度動力學角度揭示多任務RRG優化中的核心瓶頸，提出CAME-Grad優化器有效解決梯度方向衝突與幅度衰減的雙重困境。該方法作為一種骨幹無關的通用優化策略，顯著提升多種RRG模型的臨床診斷效能，並保持語言生成質量。儘管實驗結果令人鼓舞，但方法仍依賴多個超參數調整，且臨床效能評估主要基於自動標註工具，缺乏直接臨床專家評價。未來工作將致力於拓展至更多醫學影像報告生成任務及臨床實驗驗證。

**Research Gap:** 本論文雖提出理論驅動的優化方法並驗證其效果，但仍存在若干不足。首先，臨床效能評估依賴自動標註器如CheXbert，缺乏臨床專家直接評審，限制結果的臨床可信度。其次，方法引入多個超參數，對不同數據集和模型需細緻調整，增加實際部署難度。第三，SDE理論分析與實際優化過程的對應關係尚未完全實證，缺乏對失敗案例或泛化能力的深入探討。最後，實驗主要集中於胸部X光影像，對其他醫學影像類型及多模態場景的適用性未明確說明，限制了方法的廣泛應用潛力。

### 2. Gated DeltaNet-2: Decoupling Erase and Write in Linear Attention

- 分數：7
- arXiv: [2605.22791v1](http://arxiv.org/abs/2605.22791v1)
- 發表日期：2026-05-21
- 分類：cs.AI
- 入選理由：+2 提及頂級機構：mit; +3 收錄於 Hugging Face Daily Papers; +2 提及程式碼可用; LLM rank #2 / 8/10: Novel linear attention mechanism decoupling erase and write gates, showing strong empirical gains on large-scale language modeling and retrieval tasks; code available and from a reputable lab.

**Intro:** Transformer 架構因自注意力機制能直接存取歷史資訊，成為大型語言模型的主流，但其計算成本隨序列長度呈二次方增長，限制長上下文訓練與高吞吐量推理。線性注意力透過固定大小的遞迴狀態取代無界的注意力矩陣，將序列混合時間降為線性，解碼記憶體需求恆定。然而，如何在有限狀態中有效遺忘與寫入，避免干擾與關聯錯亂，是技術挑戰。Delta-rule 模型透過先讀取再寫入的方式實現有針對性的覆寫，Kimi Delta Attention 則引入通道維度的衰減強化遺忘，但仍使用單一標量門控同時控制擦除與寫入。此限制影響模型對不同維度資訊的精細控制。

**Motivation:** 在工程與產品層面，固定大小狀態的線性注意力能大幅降低長序列處理的計算與記憶體成本，適合長上下文語言模型與高效推理部署。現有方法中，擦除與寫入共用單一門控參數，限制了模型對記憶中不同通道資訊的選擇性操作，導致干擾與資訊覆蓋不足。解耦這兩者的門控機制，有助提升記憶更新的靈活性與精確度，進而改善模型在長上下文推理、檢索及語言理解任務的表現，對實際應用的穩定性與效能有直接影響。

**Method:** 本文提出 Gated DeltaNet-2，將 Delta-rule 中的擦除與寫入操作分別以通道維度的擦除門 b_t 和寫入門 w_t 控制，取代以往單一標量門控。模型保留 KDA 的通道衰減機制，並在擦除與寫入兩側分別施加不同的門控，允許針對不同通道選擇性擦除舊內容與寫入新內容。此設計在數學上等價於一種快速權重更新，並導出對應的塊狀 WY 演算法，保持高效的並行訓練。模型架構中，擦除與寫入門由獨立線性投影與 sigmoid 激活產生，並結合 L2 正規化與短卷積以穩定訓練。訓練時使用 1.3B 參數模型，搭配 100B FineWeb-Edu 資料，並支持純遞迴與混合滑動窗口注意力架構。

**Result:** 在多項語言建模與零樣本常識推理基準（WikiText、LAMBADA、PIQA 等）中，Gated DeltaNet-2 在純遞迴與混合模型設定均取得最佳平均表現。合成檢索任務 RULER 的多鍵 Needle-In-A-Haystack 測試中，該模型在長上下文與多鍵干擾情境下顯著優於 Mamba-2、Gated DeltaNet、KDA 及 Mamba-3 變體。真實世界檢索任務（如 SQuAD、TriviaQA、DROP 等）中，Gated DeltaNet-2 也展現最高平均召回率。消融實驗顯示，擦除門的通道結構對性能提升貢獻最大，寫入門亦有正面影響。訓練吞吐量方面，Gated DeltaNet-2 僅較 KDA 有輕微下降，仍保持良好效率。

**Conclusion:** Gated DeltaNet-2 成功解耦了線性注意力中擦除與寫入的門控機制，提升了固定狀態記憶的編輯靈活性與精確度。此方法在多種語言理解與檢索任務中均優於現有 Delta-rule 及通道衰減模型，且保持高效的並行訓練與推理性能。該模型架構與演算法設計兼顧理論嚴謹與實務可行，對長上下文模型的工程實現與產品部署具有實際價值。限制在於實驗主要集中於特定規模（1.3B 參數）與資料集，且未明確探討模型在更大規模或多樣化場景下的泛化性與失效案例。

**Research Gap:** 本文雖在多項任務展現優異表現，但缺乏對更大模型規模及不同語言或多模態資料的驗證，限制了泛化能力的判斷。實驗中對於模型在極端長序列（超過訓練長度 4K）或異質資料的穩定性與效率未充分說明。消融分析主要聚焦門控結構，對於其他超參數如衰減範圍、卷積設計等影響較少探討。部署層面，雖提及吞吐量影響，但未深入評估實際推理延遲與硬體資源消耗。最後，模型對於失敗案例或干擾嚴重情況下的行為未有詳細分析，這些均為後續改進方向。

### 3. Swift Sampling: Selecting Temporal Surprises via Taylor Series

- 分數：5
- arXiv: [2605.22678v1](http://arxiv.org/abs/2605.22678v1)
- 發表日期：2026-05-21
- 分類：cs.CV, cs.AI
- 入選理由：+2 提及頂級機構：mit; +3 收錄於 Hugging Face Daily Papers; LLM rank #3 / 7/10: Clear practical value with a lightweight, training-free video frame selection method that significantly improves downstream tasks; well-motivated by theory and efficient with strong empirical results.

**Intro:** 長影片中大部分畫面存在高度冗餘，關鍵資訊往往集中在時間上的突變點，即視覺特徵偏離預測軌跡的時刻。現有多數影片理解模型仍採用均勻抽樣，忽略了時間結構，導致冗餘資訊過多且效率低下。本文提出Swift Sampling，一種無需訓練的影格選擇方法，利用Taylor展開預測視覺潛在空間中的特徵軌跡，並以預測誤差（Taylor residual）作為資訊量指標，選出時間上的驚喜影格。

**Motivation:** 長影片的時間冗餘使得均勻抽樣無法有效捕捉關鍵事件，且現有訓練免費方法多依賴額外視覺編碼器或需調整超參數，增加計算成本與調參難度。受人腦預測編碼機制啟發，作者希望設計一種輕量且無需額外訓練的框架，直接利用VLM內部視覺編碼器的中間特徵，透過Taylor展開捕捉時間驚喜，提升長影片理解效率與準確度。

**Method:** Swift Sampling將影片中相鄰影格的視覺潛在特徵視為平滑軌跡，利用前N個影格特徵計算速度、加速度等導數，透過Taylor展開預測下一影格特徵。以實際特徵與預測特徵間的歐氏距離作為Taylor residual，該值越大代表該影格偏離預期軌跡，資訊量越高。最後選擇局部最大殘差的影格作為關鍵影格，確保在不同時間區段均能捕捉重要驚喜。此方法無需額外模型或影片特定調參，且計算成本僅增加0.02倍。

**Result:** 在三個長影片視覺問答基準（Video-MME、MLVU、LongVideoBench）及十個下游任務中，Swift Sampling均優於均勻抽樣及多個無需訓練的基線方法。特別是在長度超過20分鐘的影片中，準確率提升最高達12.5個百分點。與其他方法相比，Swift Sampling的計算成本最低，僅為基線的1.02倍，遠低於其他方法約1.6至1.8倍的成本。結合查詢感知方法如AKS，能進一步提升效能並降低推理成本。此外，Swift Sampling在影片標註、標題生成及推理任務中表現穩健，但在需要均勻時間覆蓋的任務如計數和時間感知上表現稍弱。

**Conclusion:** 本文提出的Swift Sampling以Taylor展開的預測誤差作為時間驚喜指標，實現了一種輕量、無需訓練且高效的長影片影格選擇方法。其低計算成本及優異的性能使其可作為多種長影片理解任務的預處理模組，提升下游模型的推理效率與準確度。限制在於目前方法為查詢無關，可能選擇視覺上驚喜但語義無關的影格，且對需要均勻時間覆蓋的任務效果有限。未來可考慮結合任務敏感性調整Taylor信號，並擴展至多模態融合以提升整體影片理解。

**Research Gap:** 本文未明確說明Swift Sampling在多樣化影片類型（如快速剪輯、劇情片）及極端視覺變化場景下的泛化能力。實驗中Taylor展開階數固定為3階，對更高階導數的效益及計算成本權衡缺乏深入探討。對於查詢敏感的任務，現方法無法動態調整選擇策略，可能導致語義無關的視覺驚喜被選中。此外，對於需要均勻時間覆蓋的任務（如計數、時間感知）表現下降，顯示方法在捕捉全局時間結構方面仍有不足。消融實驗主要集中在特徵層級與空間聚合，缺少對不同VLM架構及更大規模數據集的測試。最後，部署成本與實際系統整合的挑戰未被充分討論。

### 4. More Context, Larger Models, or Moral Knowledge? A Systematic Study of Schwartz Value Detection in Political Texts

- 分數：5
- arXiv: [2605.22641v1](http://arxiv.org/abs/2605.22641v1)
- 發表日期：2026-05-21
- 分類：cs.CL, cs.AI, cs.LG
- 入選理由：+3 收錄於 Hugging Face Daily Papers; +2 提及程式碼可用; LLM rank #4 / 7/10: Systematic and nuanced study of moral value detection in political texts, revealing important insights about context and knowledge use; includes thorough experiments and code release.

**Intro:** 本論文聚焦於政治文本中Schwartz價值偵測的挑戰，因為價值表達通常隱含且依賴上下文。研究透過ValuesML/Touché ValueEval格式，探討句子層級價值偵測時，文檔上下文、檢索式道德知識與模型規模的影響。實驗涵蓋不同上下文長度（句子、視窗、全文）、有無檢索輔助、不同規模的DeBERTa編碼器及零樣本大型語言模型（LLM）。

**Motivation:** 政治文本中的價值表達往往隱晦且依賴語境，單句輸入可能不足以捕捉價值訊息。增加上下文或引入外部道德知識理論上能提升判斷準確度，但過多上下文可能帶來干擾，且不同模型對長文本處理能力不同。此外，隨著大型語言模型興起，需比較其零樣本能力與傳統監督模型的效能差異，為工程與產品落地提供實證依據。

**Method:** 研究設計包含三種上下文條件（句子、視窗、全文）與兩種知識條件（無檢索、檢索輔助），並比較DeBERTa-v3-base與large兩種監督編碼器，以及三種不同規模的零樣本指令調教LLM。檢索知識庫由58個手工策劃的道德知識塊組成，使用MiniLM嵌入與FAISS索引進行檢索。對於編碼器，檢索知識融合方式包括早期融合、後期融合與交叉注意力三種架構。評估指標以macro-F1為主，並進行多種消融與對比實驗。

**Result:** 結果顯示全文上下文能顯著提升監督編碼器的macro-F1約3.8至4.8點，但對零樣本LLM無一致幫助，甚至可能降低表現。檢索式道德知識在所有模型與上下文條件下均帶來穩定提升，尤其在早期融合架構下效果最佳。模型規模擴大（DeBERTa-base到large，LLM從12B到123B）未必帶來性能提升。不同價值類別受上下文與檢索知識影響不同，社會情境依賴或概念混淆的價值受益最大。後期融合與交叉注意力未優於早期融合。

**Conclusion:** 本研究證實價值偵測任務中，額外上下文與外部道德知識的效用依賴於模型類型、融合策略與價值類別。監督編碼器搭配全文上下文與早期融合檢索知識為較佳組合，且優於零樣本大型語言模型。結果提醒從業者不應盲目追求更大模型或更長上下文，而應根據任務特性與模型能力謹慎選擇。評估時亦應關注各價值類別表現，避免僅以整體指標判斷。

**Research Gap:** 本研究限於單一政治文本價值偵測資料集與英語語境，尚未驗證於其他語言、領域或文本類型。檢索知識庫為手工策劃且固定，未探討自動生成或動態更新知識的影響。零樣本LLM僅採用純零樣本設定，未評估少量示例提示或微調可能帶來的提升。融合架構消融未涵蓋所有可能設計，且部分融合方法可能因超參數未充分調整而未達最佳表現。實驗中對模型推理成本與部署實務挑戰未深入討論，限制了工程落地參考價值。

### 5. SegCompass: Exploring Interpretable Alignment with Sparse Autoencoders for Enhanced Reasoning Segmentation

- 分數：5
- arXiv: [2605.22658v1](http://arxiv.org/abs/2605.22658v1)
- 發表日期：2026-05-21
