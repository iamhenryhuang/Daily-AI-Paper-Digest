**Intro:** 本文探討大型語言模型（LLM）代理在執行多步工程任務時，普遍缺乏判斷任務複雜度與所需努力的能力。現有代理多採用最大上下文優先策略，導致簡單任務被過度審查，浪費大量計算資源。作者提出任務感知的執行範圍估計，旨在讓代理能判斷任務難度、所需資訊及最短可靠路徑，以節省成本。研究基於一個可控模擬環境與真實模型驗證，提出新指標與框架，並發布對應基準與工具。

**Motivation:** 在工程與軟體維護中，許多任務本質簡單，但現有LLM代理常因缺乏複雜度判斷而過度檢索與推理，造成效率低下。這種冗餘不僅增加延遲與成本，也可能因過度操作導致失敗。工程實務中，快速判斷任務難度並採取最小必要行動是常態，缺乏此能力的代理難以實際落地。本文旨在填補此缺口，提升代理在工程場景的效能與可靠性。

**Method:** 作者定義最小充分執行（minimum-sufficient execution）與代理認知冗餘比（ACRR），用以量化代理相對於理想軌跡的多餘努力。提出E3框架（Estimate, Execute, Expand），先估計任務初始狀態（難度、範圍、風險、信心），再執行最小可行路徑，最後僅在驗證失敗時逐步擴展執行範圍。估計器結合詞彙線索與環境探測，擴展階段保證即使估計失誤也能恢復成功。實驗中使用能力不變的模擬器與真實LLM代理驗證方法有效性。

**Result:** 在MSE-Bench基準上，E3達成100%成功率，較強基線節省85%成本、91%tokens與92%檔案檢視量，且比強適應性檢索基線成本低16%。冗餘最大出現在最簡單任務，E3在各層級保持低且穩定的ACRR。消融實驗顯示估計降低成本，擴展確保成功。對估計器詞彙依賴的抗性測試證明E3架構穩健。真實LLM-Case實驗中，E3為最精簡且最快速策略，成功率與基線相當，且在複雜任務中避免過度閱讀導致的失敗與資源浪費。

**Conclusion:** 本文證明LLM代理缺乏任務複雜度判斷導致資源浪費，提出E3框架以任務感知的執行範圍估計與逐步擴展，有效降低冗餘且保持成功率。模擬與真實模型實驗均支持該方法的實用性與穩健性。該工作推動工程導向AI（EGAI）理念，強調代理行為應根植於工程任務的實際需求與結構。限制在於模擬環境與估計器較簡單，真實場景下模型能力與錯誤交互尚未充分探討。

**Research Gap:** 本研究主要依賴能力控制的模擬環境與基於詞彙規則的估計器，缺乏對多樣化真實語言表達與更複雜動態依賴的全面驗證。真實LLM實驗規模有限，且在高複雜度任務中成功率呈現隨機性，顯示方法在更大規模與多樣任務上的穩定性仍待評估。此外，估計器的學習化與泛化能力未被充分研究，可能影響擴展階段的頻率與成本。成本模型雖經敏感度分析，但實際部署中計算資源限制與API速率限制等因素可能影響效果。未來需擴展基準涵蓋更多隱藏複雜性來源，並深入探討模型錯誤與執行範圍估計的交互影響。

### 2. EvoGraph-R1: Self-Evolving Multimodal Knowledge Hypergraphs for Agentic Retrieval

- 分數：5
- arXiv: [2607.12764v1](http://arxiv.org/abs/2607.12764v1)
- 發表日期：2026-07-14
- 分類：cs.CV
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：cvpr; LLM rank #2 / 8/10: Introduces a novel self-evolving knowledge graph framework for multimodal retrieval with strong theoretical framing and demonstrated improvements on multimodal QA benchmarks, addressing key limitations of static graph ap

**Intro:** 本論文聚焦於多模態大型語言模型（MLLMs）在知識檢索增強生成（RAG）中的應用，指出現有GraphRAG方法多將知識圖譜視為靜態結構，無法動態更新與迭代推理。這種靜態設計導致跨模態推理困難、無法整合新證據及缺乏多輪檢索調整，限制了知識密集型任務的表現。作者提出EvoGraph-R1框架，將知識圖譜視為動態環境，通過代理與環境交互實現圖譜的自我演化，提升檢索與推理能力。

**Motivation:** 現有多模態RAG系統在面對複雜推理時，因文本中心化的知識碎片化、靜態圖譜結構無法更新，以及單輪檢索缺乏調整能力，導致推理鏈不完整或錯誤傳播。這些問題在工程與產品層面限制了系統的靈活性與準確性，尤其在開放域和多模態場景中更為明顯。為了實現更強的多跳推理和跨模態融合，必須設計能動態演化的知識圖譜結構與檢索策略。

**Method:** EvoGraph-R1將知識檢索建模為一個馬可夫決策過程（MDP），代理在多模態知識超圖環境中觀察狀態並執行四種行動：GRAPHRETRIEVE（圖譜檢索）、WEBSEARCH（網路擴展）、GRAPHEDIT（圖譜編輯，包括插入、更新、刪除）及ANSWER（終止並回答）。這些行動使超圖結構隨推理過程動態演化，整合新證據、修正錯誤並優化結構。代理策略透過強化學習優化，平衡答案準確度與計算效率。初始超圖由文本與視覺子圖融合構建，並以多模態編碼器嵌入以支持高效檢索。

**Result:** 在多模態VQA（E-VQA、InfoSeek、OK-VQA）及文本QA（2WikiMultiHopQA、HotpotQA、Natural Questions）基準上，EvoGraph-R1均超越多種RAG、GraphRAG及強化學習增強檢索方法。文本任務F1提升最高達6.9%，多模態任務準確率提升最高7.7%。消融實驗顯示插入與網路搜尋行動對性能貢獻最大，圖譜編輯操作有效提升推理效率與準確度。低資源實驗證明本方法在知識庫規模受限時仍保持優勢。結構與語義指標分析表明圖譜經演化後更密集且語義一致，生成答案在相關性、正確性及多樣性上均優於基線。

**Conclusion:** 本論文提出的EvoGraph-R1框架成功將知識圖譜視為動態MDP環境，通過代理驅動的多輪檢索與圖譜編輯實現自我演化，顯著提升多模態及文本問答任務的推理能力與效率。實驗結果驗證了自我演化知識圖譜在跨模態推理與開放域檢索中的價值，為未來檢索增強生成系統提供了新的設計思路。該方法在多種場景下均展現出良好的泛化性與魯棒性。

**Research Gap:** 全文未明確說明在更大規模或更複雜多模態資料集上的表現與擴展性，缺乏對系統在實際部署中計算資源消耗與延遲的詳細評估。強化學習策略的訓練穩定性與收斂速度未充分探討，可能影響實際應用的可行性。此外，對於失敗案例、錯誤類型及其對最終答案影響的分析不足，限制了對系統弱點的理解。圖譜演化過程中如何避免錯誤累積及保證長期一致性亦未深入討論。最後，WEBSEARCH行動依賴外部搜尋引擎，可能帶來外部依賴風險與安全性問題，全文未涉及相關風險評估。

### 3. MaxSAT-Based Feedback for Guiding Vision-Language Models in Sudoku

- 分數：5
- arXiv: [2607.12711v1](http://arxiv.org/abs/2607.12711v1)
- 發表日期：2026-07-14
- 分類：cs.AI, cs.LO
- 入選理由：+3 提及頂級會議：acl; +2 提及程式碼可用; LLM rank #3 / 7/10: Presents a credible neuro-symbolic integration of MaxSAT with vision-language models for Sudoku solving, showing practical gains in logical consistency and solution rates with clear evaluation.

**Intro:** 本文探討了Vision-Language Models（VLMs）在結構化視覺推理任務中，尤其是數獨遊戲解題時，缺乏邏輯一致性保障的問題。雖然VLM具備強大的感知能力，但其生成的解答常違反數獨的基本約束條件。為解決此問題，作者提出結合神經網絡與符號推理的混合方法，利用最大可滿足性問題（MaxSAT）作為一致性驗證與解答優化的工具。該方法將VLM生成的候選解作為軟約束，數獨規則作為硬約束，通過MaxSAT求解器找出最大一致子集，並將不一致部分轉化為結構化的文字與視覺反饋，指導模型進行修正。實驗在多種公開與封閉VLM上進行，結果顯示MaxSAT反饋能有效提升邏輯一致性與解題成功率。此研究展示符號優化技術在提升視覺語言推理可靠性上的潛力。

**Motivation:** VLM在結構化視覺推理中表現出色，但缺乏明確的邏輯約束機制，導致解答不一致且難以保證正確性。傳統符號方法如SAT和MaxSAT能提供形式化的正確性保證，但缺乏感知靈活性。將兩者結合可彌補各自不足，提升系統的穩健性與解題能力。對工程與產品而言，這種神經符號混合架構能在保持模型靈活性的同時，確保輸出結果符合嚴格的邏輯規則，對於需要高可靠性的視覺推理應用尤為重要。此外，透過結構化反饋機制，能有效引導模型修正錯誤，提升用戶體驗與系統可解釋性。

**Method:** 核心方法是將數獨問題以布林變數表示，並將數獨規則編碼為硬約束的CNF子句。VLM從數獨圖像中生成候選數字位置，這些候選解被編碼為軟約束子句。利用部分MaxSAT求解器，在滿足所有硬約束的前提下，最大化軟約束的滿足數量，找出最大一致的候選解子集。若存在不一致，求解器會標示衝突的候選解，並將其轉化為文字與視覺反饋，回饋給VLM以指導後續修正。實驗中採用兩種互動模式：逐步放置（每次提出一個候選解）與全盤修正（一次提出完整解盤），並在每次迭代中利用MaxSAT反饋進行優化。此流程確保神經模型負責感知與候選生成，符號求解器負責邏輯驗證與優化。

**Result:** 實驗在包含200個數獨題目的公開數據集上進行，涵蓋兩種難度等級。評估三種VLM：兩個開源模型Molmo2與Qwen3-VL，以及封閉訪問的GPT-5.5。結果顯示，未使用MaxSAT反饋時，VLM生成的解答邏輯一致性不足，解題率與完整度均有限。引入MaxSAT反饋後，所有模型的解題率與完整度均有顯著提升，尤其是GPT-5.5在全盤修正模式下，解題率從22.5%提升至36.5%，完整度從72.0%提升至80.7%。難度較高的題目改善更為明顯。開源模型雖提升幅度較小，但仍證明符號反饋對修正不一致解答有效。實驗同時證明MaxSAT求解器能快速處理數獨約束，且反饋機制能有效引導VLM修正錯誤。

**Conclusion:** 本文提出一種基於MaxSAT的神經符號框架，有效結合VLM的感知能力與符號求解器的邏輯一致性保障，用於數獨等結構化視覺推理任務。實驗結果證明，該方法能顯著提升VLM生成解答的邏輯一致性與解題成功率，尤其在全盤修正模式下效果最佳。此架構展示了神經模型與符號優化技術的互補優勢，為提升視覺語言推理系統的可靠性提供了可行路徑。限制方面，方法依賴於VLM初始解答質量，對於初始解答較差的模型提升有限，且全文未明確說明在更複雜或其他類型視覺約束問題上的泛化能力。未來工作可探索更緊密的神經符號整合及擴展至其他視覺約束任務。

**Research Gap:** 本文主要聚焦於數獨問題，缺乏對其他結構化視覺約束問題的驗證，泛化能力尚未充分評估。實驗中VLM初始解答質量對最終效果影響顯著，對於低質量初始解答的模型，MaxSAT反饋提升有限，顯示方法對神經模型性能依賴較大。全文未明確說明反饋機制在實際部署時的計算成本與延遲影響，尤其在全盤修正模式下可能存在效率瓶頸。此外，缺乏對失敗案例的深入分析，無法全面理解方法在何種情況下失效。消融實驗與其他符號推理方法的比較亦較少，限制了對方法優劣的全面評估。最後，反饋的自然語言與視覺呈現對用戶理解與模型調整的具體影響未充分探討。

### 4. ViHoRec: A Quality-Controlled Vietnamese Hotel Recommendation Dataset and Cold-Start Benchmark

- 分數：4
- arXiv: [2607.12946v1](http://arxiv.org/abs/2607.12946v1)
- 發表日期：2026-07-14
- 分類：cs.IR, cs.AI
- 入選理由：+2 提及頂級機構：mit; +2 提及程式碼可用; LLM rank #4 / 7/10: Provides a high-quality, privacy-preserving Vietnamese hotel recommendation dataset with reproducible construction and a realistic cold-start benchmark, filling a notable gap in low-resource recommendation research.

**Intro:** 本論文提出ViHoRec，一個針對越南市場的飯店推薦資料集，包含18,267筆使用者與560家飯店的互動資料。現有推薦系統研究多集中於英語市場，缺乏公開且有品質控管的越南飯店互動資料。建構此資料集面臨跨平台飯店名稱不一致、資料品質需量化審核以及隱私保護與冷啟動評測需求等挑戰。作者同時提供可重現的資料建構流程、隱私保護機制與冷啟動基準測試，填補越南推薦系統資料缺口。

**Motivation:** 推薦系統在旅遊住宿領域扮演重要角色，但越南市場缺乏公開且品質可控的飯店互動資料，限制了相關研究與應用。跨平台飯店名稱不一致導致資料難以整合，且網路爬蟲資料常有重複、缺漏與不一致問題，若無量化品質控管，難以保證資料可靠性。此外，真實場景中冷啟動問題嚴重，公開資料需支持此類評測並同時保護使用者隱私。ViHoRec旨在解決這些問題，促進越南推薦系統研究與實務落地。

**Method:** 作者從Booking.com、Traveloka與Ivivu三大越南訂房平台爬取公開評論與評分，並建立跨平台飯店實體解析機制，將不同名稱的同一飯店合併為統一標識。資料經過重複值刪除、缺失值補齊與一致性檢查，並以HMAC-SHA256加鹽方式對使用者名稱進行匿名化處理，確保隱私。資料集包含互動資料與309家飯店的內容元資料，並提供時間序列的leave-last-one-out冷啟動切分。基準測試包含多種無依賴基線模型與資料中心消融實驗，並以多種排名指標評估。

**Result:** 在公開的時間序列切分上，資料集呈現極高稀疏度（約99.5%），且近七成使用者屬於冷啟動（僅有一筆互動）。實驗結果顯示，基於潛在因子模型的BPR-MF在冷啟動使用者上表現明顯下降（Recall@10從0.120降至0.065），而基於鄰居的UserKNN在所有使用者群組中表現最佳。跨平台實體解析提升Recall@10約3.9%。內容基礎模型因元資料覆蓋有限，整體表現不及協同過濾方法。這些結果證明ViHoRec為一個真實且具挑戰性的冷啟動推薦基準。

**Conclusion:** ViHoRec提供首個公開且有嚴謹品質控管的越南飯店推薦資料集，包含跨平台實體解析、量化品質檢查、隱私保護匿名化與冷啟動基準測試。資料集反映越南市場的稀疏與冷啟動特性，揭示潛在因子模型在冷啟動場景下的性能瓶頸，鄰居方法則相對穩健。作者釋出完整資料與程式碼，促進後續研究與實務應用。限制包括資料規模仍偏小、使用者標識近似且元資料覆蓋不足，未包含多維評分，未明確說明使用者實體解析方法。

**Research Gap:** 本研究資料規模有限，僅18,267筆互動，且資料稀疏度高，可能限制模型泛化能力。使用者標識基於顯示名稱，存在多個使用者合併風險，影響使用者數量與互動次數的準確性。內容元資料僅涵蓋約53%飯店，限制內容基礎模型的應用。評分為單一整體分數，缺乏多維度評分資訊。實體解析僅針對飯店名稱，未針對使用者身份進行解析，可能影響冷啟動分析。消融實驗與基線模型雖涵蓋多種方法，但缺乏深度學習或混合模型的更全面比較。全文未明確說明資料收集過程中可能的偏差與失敗案例，亦未討論部署成本與實際應用風險。

### 5. Hy-Embodied-VLM-1.0: Efficient Physical-World Agents

- 分數：4
- arXiv: [2607.12894v1](http://arxiv.org/abs/2607.12894v1)
- 發表日期：2026-07-14
- 分類：cs.CV
- 入選理由：+2 提及頂級機構：mila, tencent; +2 提及程式碼可用; LLM rank #5 / 7/10: Proposes an efficient embodied multimodal foundation model with a clear capability taxonomy and extensive benchmark evaluation, demonstrating strong performance and practical deployment considerations.

**Intro:** 本文針對物理世界中具備行動能力的智能體，提出一款名為Hy-Embodied-VLM-1.0的高效能基礎模型。此模型不僅強調多模態感知與理解，更聚焦於行動推理、環境適應與長期互動能力。作者定義了三層次的行動能力分類法，涵蓋狀態理解、行動轉換推理及序列與適應性推理，並以此為指導設計資料管線與訓練策略。模型架構採用混合專家(MoE)結合Hy3-A3B語言骨幹與Hy-ViT2視覺編碼器，兼顧推理能力與推論效率。全文透過38項基準測試驗證模型在物理世界感知、理解與推理的綜合表現。

**Motivation:** 現有多模態視覺語言模型多聚焦於靜態圖像理解，缺乏對物理世界中行動與環境變化的推理能力。物理世界智能體需能理解當前狀態、推斷行動後的環境變化，並在長期互動中調整策略，這對模型的資料設計、訓練方法及架構提出挑戰。為了實現具備實際部署潛力的高效能物理世界智能體，必須在保持推理能力的同時降低推論成本，並建立一套系統性的能力分類與評估標準，促進模型在多層次行動推理上的全面提升。

**Method:** 核心方法包括三大部分：首先，提出一套行動導向的能力分類法，分為行動相關狀態理解、行動轉換推理及序列與適應性推理，作為資料建構與評估的指導。其次，資料涵蓋預訓練、監督微調及強化學習三階段，融合深度推理、任務條件定位、社交互動、軌跡推理、因果推理及失敗修復等多元任務，並採用拒絕採樣微調強化推理能力。最後，模型架構基於Hy3-A3B混合專家語言模型與Hy-ViT2視覺編碼器，僅啟用約3B參數以兼顧效能與推論效率。訓練過程包含預訓練、監督微調、兩階段強化學習及拒絕採樣微調，逐步提升模型的行動推理與長期決策能力。

**Result:** 在38項涵蓋感知、行動推理與序列推理的基準中，Hy-Embodied-VLM-1.0 A3B版本在19項排名第一，11項排名第二，整體平均分65.6，較最強競爭對手Qwen3.6-A3B高出4.4分，較前代Hy-Embodied-0.5 MoT-2B提升8.4分。模型在物理與語義感知、空間理解及機器人相關理解等子任務均展現領先表現。在行動轉換推理與序列適應推理方面亦優於同規模模型。閉環視覺語言導航任務中，於R2R-CE指令跟隨任務達成率57.9%，SPL 54.2%，在無路徑指導的MP3D物件目標導航中亦超越同規模基線，展現強健的長期互動與決策能力。

**Conclusion:** Hy-Embodied-VLM-1.0成功整合多模態感知、行動推理與長期適應能力，成為一款高效且具備實際部署潛力的物理世界基礎模型。透過系統性的能力分類法與多階段訓練策略，模型在多項基準與閉環任務中均展現優異表現，尤其在推理效率與能力平衡方面具明顯優勢。此模型為未來物理世界智能體的研發提供了堅實基礎，促進了從靜態理解向動態行動推理的跨越。

**Research Gap:** 全文未明確說明模型在極端環境變化、異常失敗案例或跨域泛化能力的表現，缺乏對模型在真實物理部署中計算資源消耗與延遲的詳細分析。強化學習階段雖有多樣化任務，但對於多任務間潛在衝突的消融研究不足。拒絕採樣微調的效果與成本權衡未充分探討。閉環導航任務主要集中於室內環境，缺乏戶外或更複雜場景的驗證。此外，模型對於長期互動中可能出現的累積錯誤與策略調整機制描述有限，未提供失敗恢復的具體失效案例分析。這些限制提示未來研究需加強泛化能力、實際部署評估及更全面的錯誤診斷。
