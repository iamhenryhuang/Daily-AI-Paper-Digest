### 1. Sovereign Execution Brokers: Enforcing Certificate-Bound Authority in Agentic Control Planes

- 分數：0
- arXiv: [2606.20520v1](http://arxiv.org/abs/2606.20520v1)
- 發表日期：2026-06-18
- 分類：cs.CR, cs.AI, cs.DC, cs.LG
- 入選理由：LLM rank #1 / 9/10: Introduces a novel, practical runtime enforcement mechanism (Sovereign Execution Broker) with a concrete prototype and thorough evaluation, addressing a critical security gap in autonomous agent control.

**Intro:** 隨著自主代理系統越來越多地介入雲端與基礎設施的控制流程，傳統基於身份的存取控制無法有效防止非確定性推理過程中可能產生的未授權變更。現有的保證層雖能認證提案，但缺乏在變更執行時強制執行認證權限的機制。本文提出Sovereign Execution Broker（SEB），作為一個運行時邊界，專門用於執行基於證書的代理控制，確保只有經過認證的操作能被執行，並且能即時檢查撤銷狀態與狀態漂移。

**Motivation:** 自主代理因其非確定性特性，若持有長期憑證，可能因推理錯誤或惡意攻擊導致破壞性操作。現有的IAM系統僅授權身份，無法保證操作本身的安全性，且Sovereign Assurance Boundary（SAB）雖能認證提案，但缺乏執行時的強制機制，導致代理可繞過認證直接操作。為解決此安全與操作風險，SEB作為唯一持有變更權限的實體，強制所有變更必須通過證書驗證與狀態檢查，避免代理持有長期憑證並防止重放攻擊與狀態漂移問題。

**Method:** SEB設計為一個中介執行代理，接收SAB簽發的證書與執行請求，依序驗證證書簽章、請求與證書契約匹配、有效期限、政策版本、撤銷狀態、目標基礎設施的即時狀態漂移及重放保護。通過驗證後，SEB動態鑄造範圍限定的短期執行憑證，並透過雲端API或Kubernetes TokenRequest執行變更，同時將決策與執行結果簽名記錄於不可變日誌。此架構要求基礎設施拒絕非SEB憑證的變更請求，確保執行權限集中且可撤銷。

**Result:** 作者實作了基於Go語言的SEB原型，部署於AWS與Kubernetes環境，並進行多項評測。結果顯示，SEB在Kubernetes工作負載中增加約28毫秒延遲，在AWS安全組變更中增加約137毫秒延遲，主要瓶頸為即時狀態查詢與憑證鑄造。系統能有效防止重放攻擊、撤銷繞過與狀態漂移導致的非法執行，且在模擬故障與安全攻擊時均能保持拒絕未授權操作。吞吐量方面，Kubernetes適配器可達820請求/秒，AWS適配器受限於API節流約240請求/秒。

**Conclusion:** 本文提出的SEB架構有效填補了自主代理控制平面中提案認證與執行強制之間的安全缺口。通過動態鑄造範圍限定的短期憑證，結合撤銷與狀態漂移檢查，SEB確保只有經過認證且符合當前政策與狀態的操作能被執行。原型系統在實際雲端環境中展現出可接受的延遲與良好的安全性，適合用於高風險低頻率的基礎設施變更場景。限制包括對雲端IAM系統的信任、憑證鑄造延遲、代理單點故障風險及多雲環境整合挑戰。

**Research Gap:** 本文雖完整描述SEB設計與實作，但仍有若干不足。首先，對於高頻低延遲的控制迴路不適用，缺乏針對此類場景的輕量化方案。其次，狀態漂移檢查依賴雲端API的最終一致性，可能存在TOCTOU窗口，全文未明確提出緩解策略。多雲與多帳號環境下的統一撤銷與憑證管理挑戰未充分探討。此外，代理失誤或惡意提案的語意安全性完全依賴SAB，SEB本身無法防範。最後，系統對於代理繞過SEB的防護高度依賴基礎設施配置，實際部署複雜度與運維成本未深入分析。

### 2. Confidence-Aware Automated Assessment of Student-Drawn Scientific Models

- 分數：0
- arXiv: [2606.20264v1](http://arxiv.org/abs/2606.20264v1)
- 發表日期：2026-06-18
- 分類：cs.AI
- 入選理由：LLM rank #2 / 8/10: Presents a confidence-aware automated scoring framework for student scientific drawings with practical classroom applicability and credible experimental validation.

**Intro:** 學生繪製的科學模型在科學教育中被用來評估學生的概念理解，但這類繪圖的評分通常需要專家人工判斷，導致大規模實施成本高昂且難以持續。自動化評分技術能夠提升效率，但現有方法多僅提供單一分數，缺乏對預測可靠性的判斷，限制了其在教學決策中的實用性。本文針對此問題，提出一種基於Vision Transformer的自動評分框架，結合置信度估計以支持選擇性自動評分，從而在保證評分準確性的同時減少人工負擔。

**Motivation:** 科學模型繪圖評分涉及複雜的視覺信息解讀，人工評分耗時且專業門檻高，難以在日常教學中大規模推廣。現有自動評分系統缺乏置信度指標，無法有效區分可信與不可信的預測結果，導致教師難以判斷何時應該依賴自動評分，何時需人工介入。引入置信度感知機制能夠提升系統的實用性和信任度，支持教學中更靈活且安全的評分流程。

**Method:** 本文採用預訓練的Vision Transformer作為骨幹網絡，利用低秩適配（LoRA）進行參數高效的任務特定微調。通過對輸入圖像進行語義保持的多重測試時擾動（如裁剪、旋轉），模型生成多個預測分布，並計算這些分布的平均作為最終預測。置信度定義為該平均分布中最大概率值，反映模型對該預測的穩定性。此外，提出測試時選擇機制，根據每個擾動視圖的決策確定性篩選可靠預測，進一步提升置信度估計的準確性。最後，根據置信度閾值實現選擇性自動評分，高置信度響應自動評分，低置信度響應則交由人工審核。

**Result:** 在六個符合NGSS標準的中學科學模型評估項目上，該方法相較於未微調的ViT和僅LoRA微調模型，在準確率、Cohen’s κ、精確率、召回率和F1分數等指標上均有提升。特別是CA-Selective方法通過篩選測試時視圖，進一步提高了評分一致性和穩定性。置信度分數與預測準確率呈顯著正相關，證明置信度可作為自動評分可信度的有效指標。與大型多模態基礎模型相比，該方法在推理速度和資源消耗上更具優勢，適合教室環境部署。

**Conclusion:** 本文提出的置信度感知自動評分框架有效提升了學生科學模型繪圖的評分可靠性，並通過置信度指標支持選擇性自動評分，實現了效率與準確性的平衡。實驗結果顯示該方法在多個NGSS對齊的評估項目中均有良好表現，為教育評估系統的可信部署提供了實用參考。然而，數據集僅涵蓋美國東北部中學，且評分標準依賴專家標註，可能存在地域和標註偏差。未來工作將探索更多元的學生群體和評估場景，並嘗試結合多模態數據和更細粒度的反饋。

**Research Gap:** 本文未明確說明模型在不同學生群體和多樣化教育環境中的泛化能力，數據集地域限制可能影響模型適用範圍。置信度估計雖然與準確率相關，但缺乏對置信度錯誤判斷（如高置信度錯誤預測）的詳細分析，可能影響實際應用的安全性。實驗中未展示對比其他先進置信度估計方法或消融分析，對測試時擾動類型和數量的敏感性也未充分探討。此外，推理成本雖低於大型多模態模型，但多視圖推理仍增加延遲，對實時評分場景可能構成挑戰。最後，缺乏對失敗案例的深入分析，限制了對模型局限性的全面理解。

### 3. How Do Instructions Shape Speech? Cross-Attention Attribution for Style-Captioned Text-to-Speech

- 分數：0
- arXiv: [2606.20532v1](http://arxiv.org/abs/2606.20532v1)
- 發表日期：2026-06-18
- 分類：cs.AI
- 入選理由：LLM rank #3 / 8/10: First study applying cross-attention attribution to speech diffusion models, providing novel insights into style conditioning with clear empirical evidence.

**Intro:** 本論文探討自然語言風格說明如何影響語音合成系統的聲學輸出，尤其聚焦於跨注意力機制在風格標註文本轉語音（TTS）模型中的作用。隨著TTS系統從固定說話者嵌入轉向自由形式的自然語言條件，理解單詞如何影響生成語音變得關鍵。作者首次將圖像生成領域的DAAM跨注意力歸因方法，適配並應用於語音擴散模型CapSpeech，揭示風格詞彙如何塑造聲波。此研究填補了語音生成中跨注意力解釋性不足的空白。

**Motivation:** 現代TTS系統透過自然語言描述控制語音風格，提升表達力，但缺乏對單詞如何具體影響聲學特徵的理解，限制了故障診斷與可控性提升。圖像生成領域已有跨注意力歸因技術，但語音的時間性與風格與內容的全局與局部屬性差異，使得直接套用困難。針對這一挑戰，本文提出方法以量化風格詞彙在生成過程中跨注意力的時間分布與層次動態，對工程與產品開發中風格控制的可解釋性與調試提供實用洞見。

**Method:** 作者基於CapSpeech架構，結合T5編碼器將風格說明轉為語義豐富的向量，並利用CLAP編碼器提供聲學風格標籤。核心生成模型為25層Transformer的流匹配擴散模型，通過24個ODE步驟逐步生成mel頻譜。跨注意力機制在每層每步驟中將語音查詢與風格詞鍵值匹配，作者在此過程中攔截跨注意力矩陣，計算每個詞對時間軸的注意力熱圖。進一步將詞彙分為風格、內容與功能三類，並定義時間方差、峰值比、熵等指標分析注意力分布與與聲學特徵（F0與能量）的相關性，揭示跨注意力的層與步驟動態。

**Result:** 實驗涵蓋3600組風格說明與文本組合，分析超過5萬詞彙實例。結果顯示風格詞彙的注意力時間方差顯著低於內容與功能詞，表明其作為全局調節因子分布均勻；風格詞注意力與F0及能量呈中度正相關，且語義與聲學特徵匹配良好（如“loud”與能量相關係數0.64）；跨注意力在生成初期ODE步驟與深層Transformer層達到峰值，層17注意力熵最低，對應最大選擇性，顯示模型在此階段對風格詞彙聚焦最強。功能詞則在後期步驟重要性提升，反映細節調整。這些發現首次量化了自然語言風格指令在擴散TTS模型中的跨注意力機制作用。

**Conclusion:** 本研究首次將跨注意力歸因方法引入風格標註TTS，揭示風格詞彙作為全局調節器在時間與層次上的分布特性，並與聲學特徵建立語義對應。結果支持跨注意力在流匹配擴散模型中不僅負責語音內容對齊，更承擔風格調控功能，對提升TTS系統的可解釋性與可控性具有實際意義。限制在於僅分析單一模型與有限風格詞彙，未涵蓋自然用戶輸入及多模型比較，未進行因果干預驗證。未來可擴展至更多架構、自然語料及細粒度頭部分析。

**Research Gap:** 本論文分析集中於CapSpeech模型與120組風格詞彙，缺乏對其他流匹配或擴散TTS架構的驗證，限制結果泛化。實驗使用合成風格說明，未涵蓋自然用戶輸入，可能影響實際應用場景的代表性。未進行跨注意力因果干預（如注意力編輯）以驗證歸因因果性，且未分析多頭注意力中不同頭部的專門化功能。此外，未報告模型在真實部署中計算成本與延遲影響，缺乏失敗案例與泛化能力的討論。這些缺口限制了對跨注意力機制全面理解及其工程應用的深入指導。

### 4. An Infrastructure-less, Control-Independent Solution to Relative Localisation of a Team of Mobile Robots using Ranging Measurements

- 分數：0
- arXiv: [2606.20365v1](http://arxiv.org/abs/2606.20365v1)
- 發表日期：2026-06-18
- 分類：cs.RO, cs.MA
- 入選理由：LLM rank #4 / 8/10: Proposes a fully decentralized, infrastructure-less multi-robot localization method robust to partial connectivity and unobservable conditions, addressing practical robotics challenges.

**Intro:** 本論文探討多機器人團隊在無固定基礎設施且無需控制機器人運動的情況下，如何利用量測距離進行相對定位。多機器人定位在無結構環境及協同控制任務中極為重要，但現有方法多依賴固定錨點或運動控制以確保系統可觀測性。作者提出一種去中心化、無錨點且不依賴運動控制的定位演算法，僅利用局部里程計、稀疏的機器人間距離量測及短距離通訊，這些條件在實務中較易取得。該方法採用多假設貝氏框架，維持所有可行解以應對暫時不可觀測狀態，並透過資訊共享提升部分連接網路中各機器人的定位精度。

**Motivation:** 多機器人系統在工業自動化及動態環境探索中需求可靠定位，但固定錨點難以部署且運動控制限制機器人行為，降低任務靈活性。現有方法多依賴額外感測器或嚴格運動策略以確保可觀測性，增加系統複雜度與部署成本。因此，開發一種不依賴基礎設施且不需控制運動的定位方法，能大幅降低部署門檻並提升系統適用性與擴展性，對工程與產品落地具實質意義。

**Method:** 核心方法為多假設貝氏去中心化合作定位演算法（MHDCL），每個機器人以自身參考系估計其他機器人相對位姿。演算法利用粒子濾波器表示多重假設，初始時根據首次距離量測隨機生成粒子，後續透過里程計預測與距離量測更新粒子權重並重採樣。為降低計算負擔，使用混合高斯-馮米斯分布模型將粒子聚類成多個假設。機器人間透過通訊共享聚類資訊與運動向量，允許部分連接網路中資訊傳遞與估計更新。演算法不需控制機器人運動以確保可觀測性，能在暫時不可觀測狀態下維持多重假設，反映不確定性。

**Result:** 實驗包含三種場景：一為三機器人沿曲線行走且相對位置近似共線，展示演算法在弱可觀測條件下仍能維持合理估計；二為兩機器人平行移動，第三機器人做正弦運動，證明資訊可透過運動多樣性傳遞，提升整體定位精度；三為五機器人部分連接網路，模擬結果顯示即使部分機器人無直接量測，仍能透過資訊共享維持良好定位。評估指標包括真實位姿的機率、與最可能估計的誤差及聚類區域面積，結果顯示演算法在多種實務條件下具備穩健性與擴展性。

**Conclusion:** 本文提出一種無基礎設施、無需運動控制的多機器人相對定位演算法，透過多假設貝氏框架維持所有可行解，克服傳統方法對可觀測性與運動限制的依賴。演算法僅需常見感測器與通訊設備，適用於異質系統與部分連接網路，具備良好擴展性與實務部署潛力。實驗與模擬驗證其在弱可觀測及部分連接條件下的定位能力。限制包括計算負擔較高、估計時效性受限及需設計下游應用以處理多解情況。未來將探討自適應粒子重採樣與計算優化，並擴展至多旋翼無人機與穿戴裝置等領域。

**Research Gap:** 本研究雖展現多假設框架在無基礎設施與無運動控制下的定位能力，但全文未明確說明在大規模機器人群中計算與通訊負擔的具體限制與解決方案。估計時效性問題因資訊交換頻率依賴量測事件，可能影響快速動態場景的定位精度，尚缺乏針對此問題的深入分析與改進策略。多假設輸出對下游任務的影響未充分探討，實務應用中如何融合多解以支援決策仍需研究。此外，實驗多以差動驅動機器人為主，對其他機器人動力學模型的適用性與泛化能力未充分驗證。最後，對於感測噪聲、通訊中斷及失敗案例的魯棒性分析不足，限制了對真實複雜環境中演算法穩健性的全面評估。

### 5. Optimal Order of Multi-Agent and General Many-Body Systems

- arXiv: [2606.20485v1](http://arxiv.org/abs/2606.20485v1)
