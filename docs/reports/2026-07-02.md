### 1. UniCoder: Unified Visual-to-Code Generation via Symbolic Rewards and Reference-Guided Code Optimization

- 分數：7
- arXiv: [2606.31732v1](http://arxiv.org/abs/2606.31732v1)
- 發表日期：2026-06-30
- 分類：cs.CV
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：acl; +2 提及程式碼可用; LLM rank #1 / 9/10: UniCoder introduces a novel RL framework with symbolic attribute alignment and reference-guided code optimization, achieving SOTA results on multiple benchmarks with strong evidence and practical impact in visual-to-code

**Intro:** 視覺轉程式碼生成任務要求模型能將科學圖表、向量圖形及網頁截圖精確轉換為可執行的程式碼，這需要像素級的嚴格對齊。現有多模態大型語言模型（MLLM）透過監督微調（SFT）難以達成此精度，因為SFT無法有效處理細節偏差。強化學習（RL）理論上可優化結果，但在此任務中面臨獎勵信號粗糙及探索停滯兩大挑戰。本文提出UniCoder框架，透過符號屬性對齊與參考引導的程式碼優化，解決這些問題，提升視覺到程式碼的生成品質。

**Motivation:** 視覺轉程式碼生成需嚴格保證輸入視覺與輸出渲染的結構與風格一致，然而現有獎勵機制存在兩大瓶頸：一是語義獎勵（如CLIP分數）過於粗糙，無法懲罰細微元素錯誤，導致模型產生語義幻覺；二是像素級獎勵過於敏感，對渲染噪聲反應過度，造成學習困難。此外，程式碼空間稀疏且異質，導致強化學習探索階段常陷入無效軌跡，學習停滯。這些問題限制了RL在多模態程式碼生成的應用與效果。

**Method:** UniCoder採用一個統一的強化學習框架，核心包括兩個創新機制：首先，符號屬性對齊利用輕量輔助LLM解析生成程式碼，抽取離散視覺屬性（如顏色代碼、座標範圍），以元素級別計算密集獎勵，克服語義獎勵粗糙問題；其次，參考引導程式碼優化策略在低效回合中動態注入真實參考程式碼，將盲目探索轉為有導向的策略改進，避免探索停滯。整體獎勵結合執行有效性、視覺相似度及符號屬性匹配，並基於Group Relative Policy Optimization進行優化。

**Result:** 在ChartMimic、UniSVG、Design2Code及ScreenBench等多個視覺轉程式碼基準上，8B參數的UniCoder模型超越所有開源基線，並在多項指標上接近甚至超越部分商業閉源模型如GPT-5。實驗顯示符號屬性獎勵對細節對齊（位置、顏色）尤為關鍵，而參考注入策略顯著提升生成穩定性與執行率。訓練過程中，執行率、視覺獎勵及屬性獎勵均穩定提升，證明方法有效緩解了獎勵稀疏與探索停滯問題。

**Conclusion:** 本文提出的UniCoder框架通過符號屬性對齊與參考引導優化，有效解決了視覺轉程式碼生成中獎勵粗糙與探索困難的核心問題。實驗結果證明該方法能顯著提升多模態程式碼生成的精度與穩定性，並在多個公開基準上刷新開源模型表現，縮小與商業模型的差距。此框架為未來多模態精細生成任務提供了新的思路與技術基礎。

**Research Gap:** 本文雖涵蓋科學繪圖、SVG及網頁三大靜態視覺轉程式碼場景，但未涉及互動介面、動畫或需複雜運行時行為驗證的程式碼生成，限制了方法的泛化性。符號屬性獎勵依賴於可抽取且明確的視覺屬性，對於其他視覺域或更複雜屬性尚未驗證。此外，全文未明確說明在更大規模或多樣化資料集上的表現穩定性及計算成本，缺乏對失敗案例或模型在極端視覺變化下的魯棒性分析。未來可考慮擴展屬性抽取範疇及探索更高效的參考注入策略以提升實際部署可行性。

### 2. Reinforcement Learning with Metacognitive Feedback Elicits Faithful Uncertainty Expression in LLMs

- 分數：7
- arXiv: [2606.32032v1](http://arxiv.org/abs/2606.32032v1)
- 發表日期：2026-06-30
- 分類：cs.CL, cs.AI
- 入選理由：+2 提及頂級機構：meta, mila, mit; +3 收錄於 Hugging Face Daily Papers; +2 提及程式碼可用; LLM rank #2 / 9/10: RLMF presents a credible, novel RL approach to improve LLM metacognition and uncertainty calibration with strong empirical gains and practical value for trustworthy AI.

**Intro:** 大型語言模型（LLMs）在元認知能力上存在系統性缺陷，如高信心的幻覺、無法識別知識邊界及錯誤表達內部不確定性，這些問題削弱了模型的可信度與可靠性。元認知是智能的關鍵組成部分，涉及監控與調節自身認知過程。本文針對這些缺陷，提出利用元認知反饋強化學習（RLMF）來提升模型自我評估能力，進而改進模型表達不確定性的真實性。研究聚焦於忠實校準（faithful calibration, FC）問題，即使模型表達的信心與其內在不確定性一致，這是元認知本質的挑戰。全文未明確說明其他應用場景。

**Motivation:** 在工程與產品層面，LLMs在高風險領域如醫療診斷、法律諮詢等應用時，錯誤且自信的回答會帶來嚴重後果。提升模型對自身表現的判斷能力，有助於建立用戶信任並降低錯誤決策風險。現有方法多聚焦於事實校準，忽略了模型內部信念與表達信心的一致性，導致用戶難以判斷模型何時可靠。本文提出的RLMF及元認知數據選擇機制，能在後訓練階段利用模型自我評估信號，提升不確定性表達的忠實度，對實際部署與持續優化具有重要意義。

**Method:** 核心方法為強化學習與元認知反饋結合（RLMF），在偏好優化過程中，根據模型對自身表現的判斷準確度調整完成結果的優勢值，優先強化那些不僅表現良好且自我評估準確的生成。此方法引入元認知優勢縮放機制，將模型預測的忠實校準分數與實際分數比較，計算元認知信號用於調整學習信號。此外，提出元認知數據選擇策略，根據模型自評分數挑選高價值訓練樣本，優於隨機或主動學習選擇。最後，採用兩階段流程：第一階段用RLMF校準數值信心分數，第二階段透過目標重寫將數值信心映射為自然且具上下文適應性的語言不確定性表達，提升人類可理解性與多樣性。

**Result:** 在多個LLM（如Qwen3、Llama3.1-Instruct）及10個跨領域任務上，RLMF顯著提升忠實校準指標cMFG*，平均提升約29%至25%，且保持任務準確度與事實校準。與標準強化學習相比，RLMF在數值忠實校準上最高提升63%。元認知數據選擇策略在訓練樣本挑選上優於隨機及主動學習方法。語言不確定性重寫階段經人工評估，在多樣性、自然度、實用性及上下文適應性上均優於先前方法，平均勝率達96%。此外，RLMF訓練使模型自我評估能力提升，並在多種訓練資料集上展現穩健泛化能力。全文未明確說明在極端低資源或超大規模模型上的表現。

**Conclusion:** 本文提出的RLMF及元認知數據選擇為提升LLM元認知能力提供新範式，成功應用於忠實校準任務，實現數值與語言不確定性表達的真實對齊。該方法不僅提升模型自我評估能力，也促進更自然且多樣化的語言不確定性表達，對提升模型可信度及用戶信任具實際價值。兩階段設計使語言不確定性可靈活適應不同場景，減少重訓成本。研究同時提出改進的評估指標cMFG*，更準確反映忠實校準表現。限制在於實驗主要集中於中等規模模型與特定任務，且未充分探討長期訓練穩定性及成本效益，未涵蓋更廣泛的元認知能力範疇。

**Research Gap:** 本文雖展示RLMF在忠實校準上的優勢，但缺乏對模型在極端低資源環境或超大規模模型上的驗證，泛化性仍待擴展。實驗主要基於單一訓練數據集PopQA，雖有部分跨任務測試，但對多樣化任務及真實世界複雜場景的適用性未充分評估。元認知能力的提升主要聚焦於自我評估準確度，未涵蓋其他元認知面向如策略調整或錯誤預防。成本與部署風險未明確討論，尤其是RL訓練的計算負擔及可能的模式崩潰問題。語言不確定性重寫依賴外部模型，增加系統複雜度，且對用戶偏好的適應性仍需更細緻的量化分析。全文未提供失敗案例或限制情境的深入探討。

### 3. Real-Time Source-Free Object Detection

- 分數：7
- arXiv: [2606.31834v1](http://arxiv.org/abs/2606.31834v1)
- 發表日期：2026-06-30
- 分類：cs.CV, cs.AI
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：eccv; +2 提及程式碼可用; LLM rank #3 / 8/10: RT-SFOD advances source-free object detection with a novel dual-head fusion and adaptive loss, improving speed, accuracy, and model size trade-offs with solid benchmarks and code.

**Intro:** 本論文聚焦於源無需資料的物件偵測（Source-Free Object Detection, SFOD）問題，特別針對實時應用中對延遲和記憶體限制嚴格的場景，如自動駕駛、監控和機器人。現有SFOD方法多依賴於計算量大、架構複雜的兩階段或Transformer基底模型，雖然準確率高，但不適合實時部署。作者提出基於YOLOv10這種NMS-free雙頭偵測器的SFOD框架，兼顧準確率、速度與模型大小，並在多個域轉移基準上展現優異表現。

**Motivation:** 現有SFOD方法多以準確率為優先，忽略實時應用中速度和模型大小的限制，且大多基於Faster R-CNN或DETR，這些架構不適合低延遲場景。YOLOv10具備NMS-free和雙頭設計，理論上適合實時SFOD，但直接套用現有Mean-Teacher自訓練策略效果不佳，原因在於雙頭偵測器的偽標籤生成策略不理想，以及域轉移導致多尺度特徵判別力下降。解決這兩個問題對於提升實時SFOD性能至關重要。

**Method:** 作者提出RT-SFOD框架，包含兩大核心模組：一是Dual-Head Pseudo-Label Fusion（DHF），以高精度的一對一（O2O）頭預測作為錨點，選擇性融合低重疊且高置信度的一對多（O2M）頭預測，提升偽標籤召回率同時維持精度；二是Multi-scale Adaptive Representation Diversification（MARD）損失，針對多尺度PAN特徵施加變異數和協方差約束，防止域轉移導致的特徵維度崩潰，恢復特徵判別力。兩者均只在訓練階段使用，推理時不增加額外成本。

**Result:** 在Cityscapes→Foggy Cityscapes、Sim10k→Cityscapes、KITTI→Cityscapes及Cityscapes→BDD100k等多個域轉移基準上，RT-SFOD在mAP指標上較先前最先進SFOD方法提升1.4%至3.5%。同時，得益於YOLOv10高效架構，RT-SFOD在速度上達到1.3倍提升，模型參數量約為先前方法的一半。消融實驗證明DHF和MARD各自帶來顯著性能提升，且兩者結合效果最佳。RT-SFOD亦在其他YOLO及DETR雙頭偵測器上展現良好泛化能力。

**Conclusion:** 本論文首次系統性探討NMS-free雙頭偵測器於SFOD的應用，提出RT-SFOD框架有效解決偽標籤品質與特徵判別力下降兩大挑戰，實現準確率、速度與模型大小的良好平衡。實驗結果顯示該方法在多種域轉移場景中均優於現有方法，且具備良好泛化性。該框架適合實時場景部署，對工程與產品落地具有實際價值。

**Research Gap:** 論文未明確說明在更複雜或極端域轉移場景下的表現與穩定性，尤其是偽標籤噪聲較大時MARD的適應能力。實驗主要集中於特定幾個公開資料集，缺乏多樣化場景和更多類別的驗證。消融分析雖涵蓋主要模組，但對於不同偵測器架構的細節調整與潛在限制未充分探討。此外，推理階段雖無額外成本，但訓練階段的計算負擔與資源需求未詳述，可能影響大規模部署。最後，失敗案例與模型在極端條件下的魯棒性分析缺乏，限制了對方法全面性的評估。

### 4. MemLearner: Learning to Query Context memory for Video World Models

- 分數：6
- arXiv: [2606.31734v1](http://arxiv.org/abs/2606.31734v1)
- 發表日期：2026-06-30
- 分類：cs.CV
- 入選理由：+3 收錄於 Hugging Face Daily Papers; +3 提及頂級會議：eccv; LLM rank #4 / 7/10: MemLearner proposes a learning-based adaptive memory querying method for video world models, showing clear improvements on challenging occlusion and dynamic scenarios with extensive experiments.

**Intro:** 影片世界模型旨在根據用戶操作和歷史影片幀預測未來世界狀態，對於模擬真實世界動態具有重要意義。然而，這類模型面臨記憶機制不足的挑戰，導致長時間生成的場景不一致。現有方法多採用基於規則的上下文幀檢索作為記憶，但在場景遮擋和動態物體情況下泛化能力有限。本文提出MemLearner，一種基於學習的自適應上下文查詢方法，利用查詢標記連接上下文與預測標記，提升影片生成的場景一致性。作者同時收集包含遮擋和動態物體的長影片數據集，並設計多數據集訓練策略以融合渲染與真實影片數據。實驗結果顯示MemLearner在多種挑戰場景下優於現有影片世界模型。

**Motivation:** 影片世界模型缺乏有效的記憶機制，導致生成長影片時場景不連貫，這對於互動式影片生成和真實世界模擬造成限制。傳統基於規則的上下文檢索方法無法處理複雜的遮擋和動態物體，且難以適應多變的環境。為了提升模型在真實且動態場景中的泛化能力，必須設計能夠自適應查詢歷史上下文的機制。這對工程實踐和產品落地尤為關鍵，因為穩定且一致的長影片生成是多媒體應用、虛擬現實和機器人模擬等場景的基礎需求。

**Method:** MemLearner引入可學習的查詢標記（Query tokens）作為橋樑，連接上下文標記（Context tokens）與預測標記（Predicted tokens），使模型能自適應地從歷史上下文中提取有效信息。核心設計是將查詢過程整合到預訓練的影片生成模型中，避免額外從零訓練的模組，利用模型已有的視覺先驗。為提升效率，作者提出兩種策略：一是在淺層Transformer層進行查詢，深層專注生成；二是移除不必要的注意力計算，降低計算成本。訓練時採用包含遮擋和動態物體的自建渲染數據集，並結合真實世界影片，通過多數據集訓練策略分別處理不同數據的相機姿態標註，提升模型泛化能力。

**Result:** 實驗在作者自建的包含遮擋與動態物體的長影片數據集、CaM數據集及真實世界SpatialVID數據集上進行。評估指標包括FID、FVD衡量視覺質量，PSNR、LPIPS評估記憶能力。結果顯示MemLearner在所有指標上均優於多個基線方法，包括無記憶設計的DFoT、特徵壓縮的FramePack、基於點雲檢索的VMem及基於視野重疊的CaM。特別是在遮擋和動態場景中，MemLearner顯著提升場景一致性和記憶效果。消融實驗證明查詢標記與預測標記間的注意力機制對性能至關重要，且模型不依賴於上下文標記的相機姿態信息。多數據集訓練策略有效融合渲染與真實數據，提升生成影片的真實感與記憶能力。

**Conclusion:** 本文提出MemLearner，一種基於學習的自適應上下文查詢機制，成功解決影片世界模型中長期記憶不足的問題。通過將查詢機制整合到預訓練影片生成模型中，MemLearner利用視覺先驗提升場景一致性，特別在遮擋和動態物體場景中表現優異。作者同時貢獻了一個包含精確相機姿態、遮擋和動態物體的長影片渲染數據集，並設計多數據集訓練策略以兼顧真實與渲染數據優勢。實驗結果充分驗證了方法的有效性和泛化能力。限制方面，模型容量和數據規模仍需擴大以處理更複雜場景，且記憶機制尚未解決隨生成時間線性增長的成本問題，未來可探索上下文壓縮與選擇性遺忘等方向。

**Research Gap:** 本文雖有效提升影片世界模型的記憶能力，但仍存在多項不足。首先，模型容量約1B參數，對多角色複雜互動場景表現有限，生成質量在超過五個動態角色時顯著下降，顯示模型對大規模動態追蹤能力不足。其次，雖提出查詢機制降低計算成本，但記憶仍隨生成長度線性增長，缺乏上下文壓縮或選擇性遺忘策略，限制實際部署。第三，消融中未充分探討不同查詢標記數量與結構對性能的影響，且對失敗案例及泛化邊界說明不足。最後，多數據集訓練策略雖兼顧多樣性，但真實數據中缺乏足夠重訪場景，導致純真實數據訓練無法學習有效記憶，限制了方法在無標註真實場景的適用性。

### 5. Geometry-Preserving Orthonormal Initialization for Low-Rank Adaptation in RLVR

- 分數：5
- arXiv: [2606.31813v1](http://arxiv.org/abs/2606.31813v1)
- 發表日期：2026-06-30
- 分類：cs.LG, cs.AI
- 入選理由：+3 提及頂級會議：icml; +2 提及程式碼可用; LLM rank #5 / 7/10: Theoretical and empirical analysis of LoRA initialization in RLVR leads to geometry-preserving orthonormal initialization variants that stabilize training and outperform baselines on reasoning tasks.

**Intro:** 本論文探討低秩適應（LoRA）在強化學習帶有可驗證獎勵（RLVR）場景下的初始化問題。LoRA及其變體在監督式微調（SFT）中已被廣泛應用，但在RLVR中其行為和效能尚未充分理解。特別是兩種基於奇異值分解（SVD）初始化的LoRA變體PiSSA和MiLoRA，雖在SFT中表現優於標準LoRA，卻在RLVR中表現不佳且訓練不穩定。這凸顯了RLVR中低秩矩陣初始化策略的重要性與挑戰。

**Motivation:** RLVR因其需要保持策略接近參考模型，對參數更新的限制更嚴格，導致傳統SFT中有效的LoRA初始化策略在RLVR中失效。PiSSA和MiLoRA在RLVR中出現訓練崩潰和不穩定，主要源於初始化導致的梯度放大和KL散度違規。如何設計既能保留預訓練權重幾何結構又能穩定RLVR訓練的初始化方法，是提升LoRA在RLVR中應用的關鍵。

**Method:** 論文提出理論分析證明，將LoRA中B矩陣初始化為零，並對A矩陣採用正交正規化初始化，可最小化LoRA與全參數微調之間的差距，並控制更新幅度以穩定訓練。基於此，作者設計兩種幾何保留的正交初始化方法：LoRA-RLPO利用預訓練權重的主奇異向量正交基初始化A；LoRA-RLMO則利用尾部奇異向量正交基。兩者均避免奇異值縮放，保持B矩陣為零，從而減少梯度放大和KL散度違規風險。

**Result:** 在多個數學推理基準（如GSM8K、MATH500、AIME系列）上，LoRA-RLPO和LoRA-RLMO均顯著優於標準LoRA及PiSSA、MiLoRA。實驗中，PiSSA和MiLoRA表現出訓練獎勵崩潰及較高的KL散度和梯度範數，而本方法保持較低KL散度，訓練更穩定。消融實驗顯示正交性本身即提升穩定性，結合奇異向量幾何信息則進一步提升性能。額外實驗證明方法對不同模型和任務具一定泛化能力，且在監督式微調中亦有提升。

**Conclusion:** 本論文針對RLVR中LoRA初始化不穩定問題，從理論和實驗兩方面揭示了子空間選擇與奇異值縮放對訓練穩定性的影響，並提出幾何保留的正交初始化策略。該策略有效縮小LoRA與全微調的差距，控制更新幅度，提升RLVR訓練穩定性與下游任務表現。研究結果為RLVR中低秩適應的初始化設計提供了理論依據與實踐指引，促進PEFT方法在強化學習場景的應用。

**Research Gap:** 論文主要在數學推理任務和特定模型架構上驗證方法，缺乏在更廣泛RLVR任務（如自然語言生成、對話系統）和更大規模模型上的實驗。對於不同RLVR算法（如GRPO等）及多樣化獎勵設計的適用性未充分探討。雖有理論分析，但對於初始化後訓練過程中動態調整策略的研究不足，且未明確評估計算成本與部署風險。消融實驗雖有，但對比其他先進PEFT方法的全面性有限。未提供失敗案例分析，泛化能力和穩定性在實際複雜環境中的表現仍待驗證。
