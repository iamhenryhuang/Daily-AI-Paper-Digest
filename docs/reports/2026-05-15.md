### 1. SkillOps: Managing LLM Agent Skill Libraries as Self-Maintaining Software Ecosystems

- 分數：7
- arXiv: [2605.13716v1](http://arxiv.org/abs/2605.13716v1)
- 發表日期：2026-05-13
- 分類：cs.SE, cs.MA
- 入選理由：+2 提及頂級機構：mit; +3 提及頂級會議：neurips; +2 提及程式碼可用; LLM rank #1 / 9/10: SkillOps addresses a novel and practical problem of skill-library maintenance for LLM agents with credible empirical gains and low overhead, supported by clear methodology and code availability.

**Intro:** 本論文聚焦於大型語言模型代理中技能庫的持續維護問題。隨著技能庫規模擴大，技能的新增、重用與修補會導致技能庫累積技術債，影響技能的檢索、組合與執行。現有研究多關注任務執行時的技能檢索與修復，卻忽略了技能庫本身的長期健康維護。作者提出SkillOps框架，將技能視為具型別的合約，並以分層技能生態圖組織技能，實現技能庫的健康診斷與維護。此方法可作為插件層，無需修改下游代理代碼即可提升性能。

**Motivation:** 技能庫隨時間演進會產生冗餘、接口不兼容、缺失驗證器等問題，這些問題不一定立即導致單一技能失效，但會降低整體系統的可靠性與可維護性。任務時間的修復只能解決當前失敗，無法根除庫層面的缺陷。為了讓技能庫成為可持續管理的軟體資產，必須在庫時間進行系統性的健康診斷與維護，這對工程實踐與產品穩定性至關重要。

**Method:** SkillOps將每個技能定義為一個技能合約，包含前置條件、可執行操作、產出物、驗證器與已知失效模式。技能間以依賴、兼容、冗餘與替代四種有向邊連結，形成分層技能生態圖(HSEG)。系統包含任務時間循環與庫時間循環兩部分：任務時間循環負責基於合約條件檢索技能、拼接依賴與兼容邊、插入驗證器與轉接器，並在執行失敗時嘗試局部修復；庫時間循環則根據執行日誌計算技能的效用、冗餘、兼容性、失敗風險與驗證缺口等五維健康指標，並執行合併、修復、淘汰、添加驗證器與轉接器等維護操作。此框架可作為獨立插件，輸入原始技能庫輸出維護後的庫，兼容多種下游代理。

**Result:** 在ALFWorld基準上，SkillOps作為獨立代理在200技能庫規模下達到79.5%任務成功率，較最強基線LLM_Skill_Planner提升約8.8個百分點。作為插件層，SkillOps對多個檢索型基線均有穩定提升，增益介於0.68至2.90個百分點。維護過程中幾乎不消耗額外的庫時間LLM調用，且任務時間的token使用量多數情況下降或持平。隨著技能庫規模擴大及技術債密度增加，SkillOps仍保持穩定性能，明顯優於其他基線。消融實驗顯示任務時間循環與庫時間循環均為性能提升關鍵，且添加轉接器、驗證器與修復操作效果顯著。

**Conclusion:** SkillOps提出了一種系統化的技能庫維護框架，將技能視為具型別合約並以分層圖結構組織，實現了庫時間的健康診斷與維護。實驗證明該方法能有效提升任務成功率，尤其對檢索驅動的代理效果顯著，且維護成本低，無需額外LLM推理。此工作強調技能庫應被視為持續管理的軟體資產，而非靜態檢索池，對長期運行的LLM代理系統具有實際工程價值與指導意義。

**Research Gap:** 本研究依賴結構化的技能合約與部分金標PDDL參數，實際部署中可能難以獲得此類結構化信息。所用技能庫部分為半合成，且主要基於ALFWorld，缺乏更廣泛真實場景與長期運行日誌的驗證。維護循環目前採用規則驅動，無法捕捉更複雜的語義冗餘或技能衝突，缺乏深度推理能力。CGPD風險傳播機制在當前設置中未提升任務成功率，因計劃階段未充分利用驗證器信息。此外，對維護操作的計算成本、部署風險及泛化能力尚未充分評估，未明確說明在多樣化任務與動態環境下的穩健性與失敗案例。

### 2. Topology-Preserving Neural Operator Learning via Hodge Decomposition

- 分數：5
- arXiv: [2605.13834v1](http://arxiv.org/abs/2605.13834v1)
- 發表日期：2026-05-13
- 分類：cs.LG, cs.AI, cs.CG
- 入選理由：+3 提及頂級會議：icml; +2 提及程式碼可用; LLM rank #2 / 8/10: Introduces a principled, topology-preserving neural operator learning method with strong theoretical grounding and practical improvements on physical invariants, with code released.

**Intro:** 本論文探討在帶有邊界的黎曼流形上，物理場方程的解算子學習問題，強調保持拓撲結構的重要性。傳統神經算子方法多針對歐式空間，難以同時兼顧局部幾何細節與全局拓撲不變量。作者基於霍奇理論提出一種結構感知的神經算子框架，將物理場分解為拓撲主導的低頻部分與幾何主導的高頻部分，並設計雙分支架構分別處理。此方法旨在提升在複雜幾何網格上的算子學習精度與物理一致性。

**Motivation:** 在多種物理模擬中，物理場受限於拓撲與幾何兩種結構約束，傳統神經算子難以同時保證全局拓撲不變性與捕捉局部高頻動態。現有方法如圖神經網絡或頻譜方法各有缺陷，前者難以捕捉長距離依賴且忽略高階複形結構，後者則無法嚴格保持拓撲不變量。工程與產品層面，這限制了神經算子在航空、地球物理、生物醫學等領域的應用。故需一種能結合拓撲結構與幾何細節的高效神經算子架構。

**Method:** 作者提出Hodge Spectral Duality (HSD)架構，基於離散外微分計算與霍奇分解，將k階離散微分形式分解為基底空間（低頻拓撲主導）與纖維空間（高頻幾何主導）兩部分。基底分支在截斷的霍奇頻譜空間中學習，嚴格保持拓撲不變量；纖維分支則在輔助歐式網格上利用傅立葉神經算子捕捉局部高頻結構，並透過正交投影確保不破壞拓撲結構。兩分支透過Lie-Trotter型算子分裂與輕量MLP耦合，補償拓撲與幾何算子非交換性帶來的誤差。

**Result:** 在三個物理場任務（外部氣動力學、磁靜態場、多連通環面輸運）上，HSD在MSE、梯度保真度、能量保真度及拓撲一致性指標均優於五種主流神經算子基線。特別在複雜工業幾何與多連通域中，HSD有效避免過度平滑，精確捕捉局部渦旋與全局流通量。訓練效率顯著優於消息傳遞網絡，且對重網格密度具有穩定泛化能力。消融實驗證實正交投影與耦合MLP對性能提升關鍵。

**Conclusion:** 本論文提出一種基於霍奇分解的神經算子學習框架，有效分離拓撲與幾何成分，實現了在複雜黎曼流形上的高精度物理場算子學習。方法兼顧物理守恆與高頻細節，提升了工程應用中物理模擬的準確性與效率。限制在於需固定或輕微變形的幾何以利頻譜分解預處理，且對強不連續性（如激波）適用性有限。未來可探索動態幾何下的頻譜基底轉移與更廣泛物理場類型。

**Research Gap:** 本研究依賴離散霍奇拉普拉斯頻譜分解，限制了方法在幾何動態變化大或非等距變形場景的適用性，全文未明確說明如何高效處理此類情況。纖維分支使用的輔助歐式網格及傅立葉神經算子對於強不連續現象（如激波）表現未驗證，限制了方法在含激波等劇烈物理現象中的應用。實驗中雖涵蓋多種物理場，但場景仍偏向低維流形與特定物理類型，泛化至更高維或複雜物理系統尚未驗證。消融分析主要聚焦核心模組，對於不同網格拓撲結構變化的魯棒性及失敗案例缺乏深入探討。最後，方法的實際部署成本與在大規模工業應用中的可擴展性仍需進一步評估。

### 3. EVA-Bench: A New End-to-end Framework for Evaluating Voice Agents

- 分數：5
- arXiv: [2605.13841v1](http://arxiv.org/abs/2605.13841v1)
- 發表日期：2026-05-13
- 分類：cs.SD, cs.AI, cs.CL, cs.LG
- 入選理由：+3 收錄於 Hugging Face Daily Papers; +2 提及程式碼可用; LLM rank #3 / 8/10: EVA-Bench provides a comprehensive, realistic, and practical evaluation framework for voice agents with novel composite metrics and extensive empirical analysis, plus open-source release.

**Intro:** 語音代理系統透過語音對話完成任務，已廣泛應用於企業場景。然而，現有評測缺乏同時解決模擬對話真實性與全面評估語音特有失效模式的基準。本文提出EVA-Bench，一個端到端的評測框架，涵蓋動態多輪語音對話模擬與自動驗證，並引入兩個綜合指標EVA-A（準確度）與EVA-X（體驗），以評估任務完成、語音忠實度、對話進展與回應時機等多維度品質。該框架包含213個企業場景及多種語音擾動測試，支持跨架構比較。

**Motivation:** 語音代理面臨語音特有的挑戰，如語音的線性與即時性、口音與噪音變化，導致傳統文字代理評測不足以反映真實表現。模擬對話必須真實反映多輪交互與用戶行為，且需自動檢測模擬器偏差以確保評測有效性。同時，評測指標需涵蓋任務完成、政策遵守、語音輸出正確性及用戶體驗等多面向，並能公平比較不同架構系統。這些需求促使設計一套全面且可驗證的評測框架，促進語音代理技術的工程與產品落地。

**Method:** EVA-Bench採用bot-to-bot音頻對話模擬，使用高品質的用戶模擬器結合場景目標、用戶角色與決策樹，確保多輪對話的真實性與重現性。模擬過程中引入自動驗證機制，檢測用戶行為與語音內容偏差，失敗則重新生成對話。評測指標分為EVA-A（任務完成、政策忠實度、語音輸出正確性）與EVA-X（對話進展、簡潔度、回合時序），並設計pass@1、pass@k、pass^k三種統計量衡量平均、峰值與穩定表現。框架支持多種語音擾動（口音、背景噪音）以測試系統魯棒性。

**Result:** 在213場景與12套系統（涵蓋cascade、hybrid與speech-to-speech架構）評測中，無系統能同時在EVA-A與EVA-X pass@1超過0.5，顯示準確度與體驗間存在明顯權衡。峰值表現（pass@k）普遍高於穩定表現（pass^k），中位差距達0.44，暗示單次評測易高估系統可靠性。口音擾動顯著影響cascade系統準確度，背景噪音則主要影響speech-to-speech系統體驗指標。診斷分析指出關鍵實體的語音轉錄準確度與任務完成高度相關，且faithfulness指標揭示任務完成成功時仍常見政策違反與細節錯誤。

**Conclusion:** EVA-Bench提供一套結合高真實性模擬與多維度評測指標的端到端語音代理評測框架，支持跨架構系統公平比較，並揭示準確度與用戶體驗的結構性矛盾及系統在不同語音擾動下的脆弱性。該框架開源且可擴展，為語音代理的研發與部署提供了重要工具。限制包括評測依賴LLM判斷可能帶來偏差，模擬器語音較為理想化且缺乏中斷行為，場景與語言覆蓋有限，且成本與重現性受限於商業API。未涵蓋多代理協作與實際API失效等複雜生產場景。

**Research Gap:** 本文未充分驗證模擬器與真實用戶行為的差異對評測結果的影響，尤其在情緒、挫折表達與誤解恢復方面。LLM判斷器可能對同族模型存在偏好，影響評分公正性。評測場景僅限英語與三個企業領域，缺乏多語言與多文化背景測試。語音擾動測試僅涵蓋單一口音與噪音環境，泛化性有限。未考慮多代理系統、動態API失效、延遲優化策略等生產環境常見情況。評測成本較高，限制大規模頻繁評估。最後，缺乏對有害輸出與隱私洩露風險的評估，對高風險場景適用性受限。

### 4. WARDEN: Endangered Indigenous Language Transcription and Translation with 6 Hours of Training Data

- 分數：4
- arXiv: [2605.13846v1](http://arxiv.org/abs/2605.13846v1)
- 發表日期：2026-05-13
- 分類：cs.CL, cs.AI
- 入選理由：+2 提及頂級機構：mila; +2 提及程式碼可用; LLM rank #4 / 7/10: WARDEN tackles a critical low-resource indigenous language transcription and translation problem with a novel two-stage design and domain-specific enhancements, showing strong empirical results.

**Intro:** 本論文針對極度低資源的澳洲原住民語言Wardaman，提出一套名為WARDEN的系統，能在僅有6小時標註語音資料的情況下完成語音轉錄與翻譯。Wardaman語言瀕臨消失，且缺乏大量訓練資料，傳統單一模型難以有效訓練。WARDEN透過分離語音轉錄與翻譯兩階段，並結合語言學專家知識，解決低資源挑戰。

**Motivation:** 語言文件化過程中，語音轉錄與翻譯耗時且需專業人力，尤其對瀕危語言更為困難。自動語音識別與機器翻譯系統通常依賴大量標註資料，無法直接應用於Wardaman等低資源語言。因此，開發能在極少資料下仍具備良好效能的系統，對語言保存與研究落地具有重要意義。

**Method:** WARDEN系統分為兩階段：第一階段使用Whisper-large-v3模型進行語音轉錄，透過選擇與Wardaman語音系統相似的Sundanese語言作為初始化，促進模型微調效果；第二階段則利用專家整理的Wardaman-英語詞典，結合字元錯誤率(CER)與詞綴匹配的規則式詞典匹配器，將相關詞條提供給大型語言模型(LLM)作為輔助，並透過LoRA微調LLM以提升翻譯準確度。此外，翻譯階段採用資料增強策略，包含不同長度語音片段及帶有轉錄錯誤的輸入。

**Result:** 在6小時標註資料下，WARDEN在語音轉錄階段以WER指標優於多種基線模型，尤其使用Sundanese初始化較無初始化版本降低0.12的WER。翻譯階段則以BLEU分數評估，結合詞典匹配與LLM微調的方案達到12.40分，明顯優於未使用詞典或未微調的模型。消融實驗證實詞典輔助與微調對提升翻譯表現關鍵，資料增強亦有效提升結果。質性分析顯示WARDEN生成的轉錄與翻譯更貼近語料真實內容。

**Conclusion:** 本研究提出的WARDEN系統有效解決了極低資源語言的語音轉錄與翻譯問題，透過語言相似性初始化與詞典輔助LLM微調，成功在6小時資料下達成優異表現。此系統對於瀕危語言的數位保存與語言學研究具有實際應用價值，並提供一個可供後續研究與實務應用的基準。作者亦公開資料與程式碼，促進社群合作。

**Research Gap:** 本論文雖在6小時資料下展現良好成效，但尚未明確評估系統在更大語料或其他瀕危語言上的泛化能力。詞典匹配依賴規則式方法，缺乏語義層次的匹配，可能限制翻譯準確度提升空間。LLM微調成本與部署風險未充分討論，實際應用於資源受限環境可能面臨挑戰。系統對語音品質、口音變異及長篇語音的處理能力亦未深入驗證。此外，失敗案例與錯誤分析較少，限制對系統弱點的理解。全文未明確說明模型對社群需求的適應性與使用者反饋。

### 5. Neurosymbolic Auditing of Natural-Language Software Requirements

- 分數：4
- arXiv: [2605.13817v1](http://arxiv.org/abs/2605.13817v1)
- 發表日期：2026-05-13

**Intro:** 自然語言軟體需求常存在歧義、不一致及規格不足的問題，尤其在安全關鍵領域中，這些缺陷會導致形式模型驗證錯誤的規格，進而產生不安全的實作。本文提出利用大型語言模型結合SMT求解器，將自然語言需求自動轉換為形式邏輯，並透過求解器查詢檢測歧義、不一致、空洞及安全違規。作者開發了VERIMED，一個針對醫療設備軟體需求的神經符號管線，並在血液透析安全需求上進行實驗驗證。此研究聚焦於提升需求規格的嚴謹性與可審核性，對工程與產品安全性有直接影響。

**Motivation:** 自然語言需求的模糊與不一致會導致形式驗證錯誤，尤其在醫療設備等安全關鍵系統中，錯誤規格可能引發嚴重後果。現有自動形式化方法多忽略需求本身的歧義與語意正確性，缺乏對形式化結果的嚴格審核。VERIMED透過結合LLM與SMT求解器，不僅自動生成形式規格，還能檢測並定位需求缺陷，並利用求解器回饋指導需求修正，提升需求的準確性與安全性，對實際軟體開發與驗證流程具重要價值。

**Method:** VERIMED包含四個階段：首先，模糊解析器對每個需求進行多次獨立形式化，利用雙向SMT等價性檢查判斷歧義，並透過求解器提供的具體反例引導LLM澄清需求；其次，自動形式化器將澄清後的需求集轉換為統一的SMT-LIB模型，並在語法錯誤時進行有限次重試修正；第三，屬性驗證器執行四種審核查詢（全局一致性、空洞性、可違反性、冗餘性），定位結構性缺陷；最後，查詢驗證器針對場景安全問題檢驗需求模型，並利用求解器反例指導LLM修正錯誤答案。整體流程結合LLM的語言理解與SMT求解器的嚴謹邏輯推理，實現需求的神經符號審核。
