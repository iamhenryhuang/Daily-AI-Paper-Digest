# Paper ranking preferences

Prioritize:
- Agent systems, coding agents, RAG, LLM evaluation, safety, and practical model tooling.
- Papers with clear methods, measurable results, reproducible artifacts, or released code.
- Work that suggests useful implementation ideas for real products or research prototypes.
- Papers that challenge current assumptions or expose a concrete limitation in common LLM workflows.

Deprioritize:
- Incremental benchmark-only papers without a clear insight.
- Papers whose main signal is author affiliation, lab prestige, or social popularity.
- Abstracts that make broad claims without concrete evidence, baselines, or evaluation details.
